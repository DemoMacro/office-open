---
title: "形状与文本"
description: "创建带格式文本、段落、对齐、列表和文本自适应的形状"
---

形状是幻灯片的主要构建块。一个 `Shape` 可以通过 `Paragraph` 和 `TextRun` 对象来包含文本。

## 基本形状

```ts
import { Shape } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "children": [{ "shape": { "x": 1, "y": 1, "width": 5, "height": 3, "text": "Simple text" } }]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 5,
      height: 3,
      text: "Simple text",
    }),
  ],
}
```

::

## 几何预设

使用 `geometry` 选项设置形状类型：

```ts
new Shape({ geometry: "rect" }); // 矩形（默认）
new Shape({ geometry: "ellipse" }); // 椭圆/圆形
new Shape({ geometry: "roundRect" }); // 圆角矩形
new Shape({ geometry: "triangle" });
new Shape({ geometry: "diamond" });
new Shape({ geometry: "star5" });
new Shape({ geometry: "hexagon" });
new Shape({ geometry: "chevron" });
new Shape({ geometry: "arrow" });
```

## 段落和文本片段

使用 `paragraphs` 完全控制文本格式：

::api-example{type="pptx"}

```json [JSON]
{
  "children": [
    {
      "shape": {
        "x": 1,
        "y": 1,
        "width": 6,
        "height": 3,
        "paragraphs": [
          {
            "children": [{ "text": "Bold title", "bold": true, "fontSize": 28, "fill": "2E74B5" }]
          },
          {
            "children": [
              { "text": "Normal text with " },
              { "text": "italic", "italic": true },
              { "text": " and " },
              { "text": "colored", "fill": "FF0000", "underline": "SINGLE" }
            ]
          }
        ]
      }
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 6,
      height: 3,
      paragraphs: [
        new Paragraph({
          children: [new TextRun({ text: "Bold title", bold: true, fontSize: 28, fill: "2E74B5" })],
        }),
        new Paragraph({
          children: [
            new TextRun({ text: "Normal text with " }),
            new TextRun({ text: "italic", italic: true }),
            new TextRun({ text: " and " }),
            new TextRun({ text: "colored", fill: "FF0000", underline: "SINGLE" }),
          ],
        }),
      ],
    }),
  ],
}
```

::

## TextRun 格式选项

```ts
new TextRun({
  text: "Formatted text",
  fontSize: 24, // 字号（磅）
  bold: true,
  italic: true,
  underline: "SINGLE", // "SINGLE" | "DOUBLE" | "NONE"
  fill: "4472C4", // 十六进制颜色，不带 #
  font: "Calibri", // 字体
});
```

## 对齐方式

```ts
new Paragraph({
  properties: { alignment: "LEFT" }, // "LEFT" | "CENTER" | "RIGHT" | "JUSTIFY"
  children: [new TextRun({ text: "Aligned text" })],
});
```

## 项目符号列表

::api-example{type="pptx"}

```json [JSON]
{
  "children": [
    {
      "shape": {
        "x": 1,
        "y": 1,
        "width": 6,
        "height": 4,
        "paragraphs": [
          {
            "properties": { "bullet": { "type": "char", "char": "●" } },
            "children": [{ "text": "First item" }]
          },
          {
            "properties": { "bullet": { "type": "char", "char": "●" } },
            "children": [{ "text": "Second item" }]
          },
          {
            "properties": { "bullet": { "type": "autoNum", "format": "arabicPeriod" } },
            "children": [{ "text": "Numbered item" }]
          }
        ]
      }
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 6,
      height: 4,
      paragraphs: [
        new Paragraph({
          properties: { bullet: { type: "char", char: "●" } },
          children: [new TextRun({ text: "First item" })],
        }),
        new Paragraph({
          properties: { bullet: { type: "char", char: "●" } },
          children: [new TextRun({ text: "Second item" })],
        }),
        new Paragraph({
          properties: { bullet: { type: "autoNum", format: "arabicPeriod" } },
          children: [new TextRun({ text: "Numbered item" })],
        }),
      ],
    }),
  ],
}
```

## 文本锚点（垂直对齐）

控制文本在形状内的垂直位置：

::api-example{type="pptx"}

```json [JSON]
{
  "children": [
    {
      "shape": {
        "x": 1,
        "y": 1,
        "width": 5,
        "height": 3,
        "textAnchor": "TOP",
        "text": "Top-aligned text"
      }
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 5,
      height: 3,
      textAnchor: "TOP", // "TOP" | "CENTER" | "BOTTOM"
      text: "Top-aligned text",
    }),
  ],
}
```

## 文本自适应

::api-example{type="pptx"}

```json [JSON]
{
  "children": [
    {
      "shape": {
        "x": 1,
        "y": 1,
        "width": 4,
        "height": 2,
        "text": "This text will shrink to fit the shape",
        "textAutoFit": "normal"
      }
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 4,
      height: 2,
      text: "This text will shrink to fit the shape",
      textAutoFit: "normal", // 自动缩小文本以适应形状
      // 或 textAutoFit: "shape" 自动调整形状大小
    }),
  ],
}
```

## 文本分栏

::api-example{type="pptx"}

```json [JSON]
{
  "children": [
    {
      "shape": {
        "x": 1,
        "y": 1,
        "width": 8,
        "height": 4,
        "textColumns": 2,
        "textColumnSpacing": 0.5,
        "paragraphs": [
          { "children": [{ "text": "Column 1 content..." }] },
          { "children": [{ "text": "Column 2 content..." }] }
        ]
      }
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 8,
      height: 4,
      textColumns: 2,
      textColumnSpacing: 0.5,
      paragraphs: [
        new Paragraph({ children: [new TextRun({ text: "Column 1 content..." })] }),
        new Paragraph({ children: [new TextRun({ text: "Column 2 content..." })] }),
      ],
    }),
  ],
}
```
