title: "Headers and Footers"
description: "Add dates, slide numbers, and custom text to slide footers"
---

Use `headerFooter` to add consistent metadata across slides — dates, slide numbers, and custom text.

## Slide-Level Header/Footer

Apply headers and footers on individual slides:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "headerFooter": {
        "dateTime": true,
        "footer": "Confidential",
        "slideNumber": true
      },
      "children": [
        { "shape": { "x": 50, "y": 50, "width": 800, "height": 350, "text": "Content slide" } }
      ]
    }
  ]
}
```

```ts [Class]
import { Presentation, Slide, Shape } from "@office-open/pptx";

const pres = new Presentation({
  slides: [
    {
      headerFooter: {
        dateTime: true,
        footer: "Confidential",
        slideNumber: true,
      },
      children: [
        new Shape({
          x: 50,
          y: 50,
          width: 800,
          height: 350,
          text: "Content slide",
        }),
      ],
    },
  ],
});
```

::

## HeaderFooter Options

| Property      | Type                | Description                                |
| ------------- | ------------------- | ------------------------------------------ |
| `dateTime`    | `boolean`           | Show/hide date/time placeholder            |
| `footer`      | `string \| boolean` | Footer text, or `true` to show placeholder |
| `slideNumber` | `boolean`           | Show/hide slide number                     |
| `header`      | `boolean`           | Show/hide header placeholder               |

## Slide-Level Override

Override the header/footer on individual slides:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "headerFooter": {
        "footer": "Draft — Not for Distribution",
        "slideNumber": false
      },
      "children": [
        { "shape": { "x": 50, "y": 50, "width": 800, "height": 350, "text": "Special slide" } }
      ]
    }
  ]
}
```

```ts [Class]
{
    children: [new Shape({ x: 50, y: 50, width: 800, height: 350, text: "Special slide" })],
    headerFooter: {
        footer: "Draft — Not for Distribution",
        slideNumber: false,
    },
}
```

## Full Example

::api-example{type="pptx"}

```json [JSON]
{
  "title": "Quarterly Report",
  "creator": "Finance Team",
  "slides": [
    {
      "headerFooter": {
        "dateTime": true,
        "footer": "Confidential",
        "slideNumber": true
      },
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 800,
            "height": 200,
            "paragraphs": [
              {
                "properties": { "alignment": "CENTER" },
                "children": [{ "text": "Q4 2025 Report", "fontSize": 36, "bold": true }]
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import { Presentation, Slide, Shape, Paragraph, TextRun, Packer } from "@office-open/pptx";
import * as fs from "node:fs";

const pres = new Presentation({
  title: "Quarterly Report",
  creator: "Finance Team",
  slides: [
    {
      headerFooter: {
        dateTime: true,
        footer: "Confidential",
        slideNumber: true,
      },
      children: [
        new Shape({
          x: 50,
          y: 50,
          width: 800,
          height: 200,
          paragraphs: [
            new Paragraph({
              properties: { alignment: "CENTER" },
              children: [new TextRun({ text: "Q4 2025 Report", fontSize: 36, bold: true })],
            }),
          ],
        }),
      ],
    },
  ],
});

const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("report.pptx", buffer);
```
