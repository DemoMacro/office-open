title: "AI 集成"
description: "通过 MCP、Skills 和 LLMs.txt 将 Office Open 与 AI 代理配合使用"
---

Office Open 文档支持多种 AI 集成方式。AI 代理可以搜索文档、安装技能包或直接消费原始 Markdown 参考。

## 文档助手

文档站点内置了由 Docus 驱动的 AI 助手，支持通过自然语言查询搜索文档内容。

- **浮动输入框** — 在任意文档页面提问（支持键盘快捷键）
- **AI 解释** — 对当前页面内容生成解释说明
- **引用来源** — 回复中包含可导航的文档页面链接
- **代码示例** — 生成可复制的代码片段

## MCP 服务器

文档站点在 `/mcp` 路径暴露了 MCP（模型上下文协议）服务器，AI 代理可连接此服务器搜索文档。

### 在 Claude Code 中使用

将 Office Open 文档 MCP 服务器添加到 Claude Code 配置中：

```json
{
    "mcpServers": {
        "office-open": {
            "url": "https://www.office-open.com/mcp"
        }
    }
}
```

### 在 Cursor / 其他 MCP 客户端中使用

在客户端的 MCP 设置中配置远程 MCP 服务器 URL `https://www.office-open.com/mcp`。该服务器提供 `search` 工具用于查询文档内容。

## Agent Skills

Skills 遵循 [Cloudflare Agent Skills Discovery RFC](https://github.com/cloudflare/agent-skills-discovery-rfc)，为 AI 代理提供关于 Office Open API 的专业知识。

### 安装 Skills

```bash
npx skills add https://www.office-open.com
```

此命令会自动发现并安装 Office Open skill，使你的 AI 代理能够访问：

- DOCX 和 PPTX 快速入门指南
- 所有包的完整 API 参考
- 度量单位转换表
- 常见代码模式

### 包含内容

Skill 按渐进式披露组织：

| 资源                     | 内容                                          |
| ------------------------ | --------------------------------------------- |
| `SKILL.md`               | 概览、快速入门、常见模式                      |
| `references/docx-api.md` | DOCX 完整 API（文本、表格、图片、数学、样式） |
| `references/pptx-api.md` | PPTX 完整 API（形状、效果、动画、填充）       |
| `references/units.md`    | 度量单位和转换函数                            |

### 手动安装

如果你的代理不支持 `npx skills add`，可以手动引用 skill 文件：

```
https://www.office-open.com/.well-known/skills/office-open/SKILL.md
```

## LLMs.txt

文档站点生成 `llms.txt` 和 `llms-full.txt` 文件供 LLM 消费：

- **`/llms.txt`** — 所有文档页面的简洁索引
- **`/llms-full.txt`** — Markdown 格式的完整文档内容

这些文件遵循 [llms.txt 规范](https://llmstxt.org/)，专为需要批量获取文档内容的 AI 代理设计。

### 在 CLAUDE.md 中使用

在项目的 `CLAUDE.md` 中添加：

```markdown
对于 Office Open XML API，参考：

- https://www.office-open.com/llms.txt
```

## 总结

| 方式       | 适用场景                       |
| ---------- | ------------------------------ |
| 文档助手   | 在文档站点上进行交互式问答     |
| MCP 服务器 | 在 AI 编码工具中实时搜索文档   |
| Skills     | 为 AI 代理提供持久的 API 知识  |
| LLMs.txt   | 批量获取文档内容用于上下文窗口 |
