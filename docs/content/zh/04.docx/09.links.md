---
title: "链接"
description: "Word 文档中的外部超链接、书签和交叉引用"
---

通过超链接和书签为文档添加导航功能。

## 外部超链接

链接到外部 URL：

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              {
                "text": "Visit example.com",
                "hyperlink": { "link": "https://example.com" },
                "underline": { "type": "single" },
                "color": "0563C1"
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  sections: [
    {
      children: [
        {
          paragraph: {
            children: [
              {
                text: "Visit example.com",
                hyperlink: { link: "https://example.com" },
                underline: { type: "single" },
                color: "0563C1",
              },
            ],
          },
        },
      ],
    },
  ],
});
```

::

## 内部超链接

使用 `hyperlink: { anchor }` 链接到同一文档中的书签：

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              {
                "text": "Go to Introduction",
                "hyperlink": { "anchor": "section-intro" },
                "color": "0563C1"
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  sections: [
    {
      children: [
        {
          paragraph: {
            children: [
              {
                text: "Go to Introduction",
                hyperlink: { anchor: "section-intro" },
                color: "0563C1",
              },
            ],
          },
        },
      ],
    },
  ],
});
```

::

## 书签

定义可被链接的书签：

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              { "bookmarkStart": { "name": "section-intro", "id": 0 } },
              "Introduction",
              { "bookmarkEnd": 0 }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  sections: [
    {
      children: [
        {
          paragraph: {
            children: [
              { bookmarkStart: { name: "section-intro", id: 0 } },
              "Introduction",
              { bookmarkEnd: 0 },
            ],
          },
        },
      ],
    },
  ],
});
```

::

将书签与 `hyperlink: { anchor }` 配合使用，可在文档内进行交叉引用。`anchor` 属性可以引用书签的 `name`。

## 屏幕提示

添加悬停时显示的工具提示：

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              {
                "text": "Example Link",
                "hyperlink": { "link": "https://example.com", "tooltip": "Click to visit Example" },
                "color": "0563C1"
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  sections: [
    {
      children: [
        {
          paragraph: {
            children: [
              {
                text: "Example Link",
                hyperlink: {
                  link: "https://example.com",
                  tooltip: "Click to visit Example",
                },
                color: "0563C1",
              },
            ],
          },
        },
      ],
    },
  ],
});
```

::
