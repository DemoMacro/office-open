Create a spreadsheet from scratch in three steps: define data, build a workbook, and export.

## Step 1 — Install

::code-group

```bash [pnpm]
pnpm add @office-open/xlsx
```

```bash [npm]
npm install @office-open/xlsx
```

```bash [yarn]
yarn add @office-open/xlsx
```

```bash [bun]
bun add @office-open/xlsx
```

::

## Step 2 — Create a Workbook

```ts
import { generateWorkbook } from "@office-open/xlsx";

const opts = {
  worksheets: [
    {
      name: "Sales",
      rows: [
        {
          cells: [
            { value: "Product", style: { font: { bold: true } } },
            { value: "Q1", style: { font: { bold: true } } },
            { value: "Q2", style: { font: { bold: true } } },
          ],
        },
        { cells: [{ value: "Widget" }, { value: 100 }, { value: 150 }] },
        { cells: [{ value: "Gadget" }, { value: 200 }, { value: 250 }] },
      ],
    },
  ],
};
```

Or use the JSON API:

## Step 3 — Export

```ts
// Node.js — write to file
import { writeFileSync } from "node:fs";
import { generateWorkbook } from "@office-open/xlsx";

const buffer = await generateWorkbook({
  worksheets: [
    {
      name: "Sales",
      rows: [
        { cells: [{ value: "Product" }, { value: "Q1" }, { value: "Q2" }] },
        { cells: [{ value: "Widget" }, { value: 100 }, { value: 150 }] },
        { cells: [{ value: "Gadget" }, { value: 200 }, { value: 250 }] },
      ],
    },
  ],
});
writeFileSync("output.xlsx", buffer);
```

```ts
// Browser — download as file
import { generateWorkbook } from "@office-open/xlsx";

const blob = await generateWorkbook(
  {
    worksheets: [
      {
        name: "Sales",
        rows: [
          { cells: [{ value: "Product" }, { value: "Q1" }, { value: "Q2" }] },
          { cells: [{ value: "Widget" }, { value: 100 }, { value: 150 }] },
          { cells: [{ value: "Gadget" }, { value: 200 }, { value: 250 }] },
        ],
      },
    ],
  },
  { type: "blob" },
);
const url = URL.createObjectURL(blob);
const a = document.createElement("a");
a.href = url;
a.download = "output.xlsx";
a.click();
```

## Multiple Worksheets

```ts
import { generateWorkbook } from "@office-open/xlsx";

const buffer = await generateWorkbook({
  worksheets: [
    {
      name: "Summary",
      rows: [{ cells: [{ value: "Total" }, { value: 500 }] }],
    },
    {
      name: "Details",
      rows: [
        { cells: [{ value: "Item" }, { value: "Amount" }] },
        { cells: [{ value: "A" }, { value: 200 }] },
        { cells: [{ value: "B" }, { value: 300 }] },
      ],
    },
  ],
});
```

## Output Formats

`generateWorkbook` supports multiple output formats via the optional second argument. See the [Export](./export) page for the full list.

| Usage                                        | Returns  | Use Case                |
| -------------------------------------------- | -------- | ----------------------- |
| `generateWorkbook(opts)`                     | `Buffer` | Node.js file I/O        |
| `generateWorkbook(opts, { type: "blob" })`   | `Blob`   | Browser downloads       |
| `generateWorkbook(opts, { type: "base64" })` | `string` | Data URLs, API payloads |
| `generateWorkbookSync(opts)`                 | `Buffer` | Synchronous Node.js I/O |
