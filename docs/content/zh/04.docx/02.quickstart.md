只需三步即可从零创建一个 Word 文档：定义内容、构建文档、导出。

## 第一步 — 安装

```bash
pnpm add @office-open/docx
```

## 第二步 — 创建文档

```ts
import { Document, Packer, Paragraph, TextRun } from "@office-open/docx";

const doc = new Document({
  sections: [
    {
      children: [
        // 标题段落
        new Paragraph({
          heading: "Heading1",
          children: [new TextRun({ text: "My First Document", bold: true, size: 32 })],
        }),

        // 正文段落
        new Paragraph({
          children: [
            new TextRun("This is a paragraph with "),
            new TextRun({ text: "bold", bold: true }),
            new TextRun(" and "),
            new TextRun({ text: "italic", italics: true }),
            new TextRun(" text."),
          ],
        }),
      ],
    },
  ],
});
```

## 第三步 — 导出

```ts
// Node.js — 写入文件
import { writeFileSync } from "node:fs";

const buffer = await Packer.toBuffer(doc);
writeFileSync("output.docx", buffer);
```

```ts
// 浏览器 — 下载文件
const blob = await Packer.toBlob(doc);
const url = URL.createObjectURL(blob);
const a = document.createElement("a");
a.href = url;
a.download = "output.docx";
a.click();
```

## 输出格式

`Packer` 类支持多种输出格式：

| 方法                         | 返回值           | 适用场景           |
| ---------------------------- | ---------------- | ------------------ |
| `Packer.toBuffer(doc)`       | `Buffer`         | Node.js 文件 I/O   |
| `Packer.toBlob(doc)`         | `Blob`           | 浏览器下载         |
| `Packer.toBase64String(doc)` | `string`         | Data URL、API 载荷 |
| `Packer.toString(doc)`       | `string`         | 调试、检查         |
| `Packer.toStream(doc)`       | `ReadableStream` | 流式传输大文件     |
