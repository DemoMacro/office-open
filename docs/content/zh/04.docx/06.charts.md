title: "图表"
description: "在文档中添加柱状图、折线图、饼图、面积图和散点图"
---

使用 `ChartRun` 在文档中嵌入图表。

## 基本柱状图

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              {
                "chart": {
                  "type": "column",
                  "data": {
                    "categories": ["Q1", "Q2", "Q3", "Q4"],
                    "series": [{ "name": "Sales", "values": [10, 20, 15, 25] }]
                  },
                  "title": "季度销售额",
                  "transformation": { "width": 500, "height": 300 }
                }
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import { Document, Paragraph, ChartRun } from "@office-open/docx";

{
  children: [
    new Paragraph({
      children: [
        new ChartRun({
          type: "column",
          data: {
            categories: ["Q1", "Q2", "Q3", "Q4"],
            series: [{ name: "Sales", values: [10, 20, 15, 25] }],
          },
          title: "季度销售额",
          transformation: { width: 500, height: 300 },
        }),
      ],
    }),
  ],
}
```

::

## 图表类型

`type` 属性选择图表类型：

| 类型   | 值          |
| ------ | ----------- |
| 柱状图 | `"column"`  |
| 条形图 | `"bar"`     |
| 折线图 | `"line"`    |
| 饼图   | `"pie"`     |
| 面积图 | `"area"`    |
| 散点图 | `"scatter"` |

## 多系列图表

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              {
                "chart": {
                  "type": "column",
                  "data": {
                    "categories": ["Q1", "Q2", "Q3", "Q4"],
                    "series": [
                      { "name": "产品 A", "values": [10, 20, 15, 25] },
                      { "name": "产品 B", "values": [15, 10, 20, 30] }
                    ]
                  },
                  "transformation": { "width": 500, "height": 300 }
                }
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
new ChartRun({
  type: "column",
  data: {
    categories: ["Q1", "Q2", "Q3", "Q4"],
    series: [
      { name: "产品 A", values: [10, 20, 15, 25] },
      { name: "产品 B", values: [15, 10, 20, 30] },
    ],
  },
  transformation: { width: 500, height: 300 },
});
```

## 散点图

散点图使用 `xValues` 和 `yValues` 而非 `categories`：

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              {
                "chart": {
                  "type": "scatter",
                  "data": {
                    "categories": [],
                    "series": [
                      {
                        "name": "数据集 1",
                        "values": [],
                        "xValues": [1, 2, 3, 4, 5],
                        "yValues": [2, 4, 1, 5, 3]
                      }
                    ]
                  },
                  "transformation": { "width": 500, "height": 300 }
                }
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
new ChartRun({
  type: "scatter",
  data: {
    categories: [],
    series: [
      {
        name: "数据集 1",
        values: [],
        xValues: [1, 2, 3, 4, 5],
        yValues: [2, 4, 1, 5, 3],
      },
    ],
  },
  transformation: { width: 500, height: 300 },
});
```

## ChartRun 选项

| 选项              | 类型                 | 说明                           |
| ----------------- | -------------------- | ------------------------------ |
| `type`            | `string`             | 图表类型（见上表）             |
| `data`            | `object`             | 图表数据（categories、series） |
| `data.categories` | `string[]`           | 类别标签                       |
| `data.series`     | `{ name, values }[]` | 系列数据                       |
| `transformation`  | `{ width, height }`  | 显示尺寸                       |
| `title`           | `string`             | 图表标题                       |
| `showLegend`      | `boolean`            | 是否显示图例（默认：`true`）   |
| `style`           | `number`             | 样式预设（2–48）               |
| `floating`        | `object`             | 浮动定位选项                   |
