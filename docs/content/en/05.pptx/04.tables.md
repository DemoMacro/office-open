Use `TableFrame` to add structured tabular data to slides.

## Basic Table

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "table": {
            "x": 50,
            "y": 50,
            "width": 800,
            "height": 300,
            "rows": [
              { "cells": [{ "text": "Name" }, { "text": "Role" }, { "text": "Department" }] },
              { "cells": [{ "text": "Alice" }, { "text": "Engineer" }, { "text": "Dev" }] },
              { "cells": [{ "text": "Bob" }, { "text": "Designer" }, { "text": "UX" }] }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import { TableFrame, Paragraph, TextRun } from "@office-open/pptx";

{
  children: [
    new TableFrame({
      x: 50,
      y: 50,
      width: 800,
      height: 300,
      rows: [
        {
          cells: [
            { text: "Name" },
            { text: "Role" },
            { text: "Department" },
          ],
        },
        { cells: [{ text: "Alice" }, { text: "Engineer" }, { text: "Dev" }] },
        { cells: [{ text: "Bob" }, { text: "Designer" }, { text: "UX" }] },
      ],
    }),
  ],
}
```

## Cell Formatting

Use `children` with `Paragraph` and `TextRun` for formatted cell content:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "table": {
            "x": 50,
            "y": 50,
            "width": 800,
            "height": 300,
            "rows": [
              {
                "cells": [
                  {
                    "fill": "2E74B5",
                    "verticalAlign": "CENTER",
                    "children": [
                      {
                        "paragraph": {
                          "children": [
                            { "text": "Header", "bold": true, "fontSize": 14, "fill": "FFFFFF" }
                          ]
                        }
                      }
                    ]
                  }
                ]
              },
              {
                "cells": [
                  {
                    "fill": "F2F2F2",
                    "children": [
                      {
                        "paragraph": {
                          "children": [{ "text": "Data cell", "fontSize": 12 }]
                        }
                      }
                    ]
                  }
                ]
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
new TableFrame({
  x: 50,
  y: 50,
  width: 800,
  height: 300,
  rows: [
    {
      cells: [
        {
          fill: "2E74B5",
          verticalAlign: "CENTER",
          children: [
            new Paragraph({
              children: [
                new TextRun({
                  text: "Header",
                  bold: true,
                  fontSize: 14,
                  fill: "FFFFFF",
                }),
              ],
            }),
          ],
        },
      ],
    },
    {
      cells: [
        {
          fill: "F2F2F2",
          children: [
            new Paragraph({
              children: [new TextRun({ text: "Data cell", fontSize: 12 })],
            }),
          ],
        },
      ],
    },
  ],
});
```

## Rich Cell Content

Cells can contain multiple paragraphs:

```ts
{
    cells: [
        {
            children: [
                new Paragraph({
                    children: [
                        new TextRun({ text: "Bold title", bold: true, fontSize: 14 }),
                    ],
                }),
                new Paragraph({
                    children: [
                        new TextRun({ text: "Description", fontSize: 11, fill: "666666" }),
                    ],
                }),
            ],
        },
    ],
}
```

## Merged Cells

Merge cells using `rowSpan` and `columnSpan`:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "table": {
            "x": 50,
            "y": 50,
            "width": 800,
            "height": 350,
            "rows": [
              {
                "cells": [{ "text": "Category", "rowSpan": 2 }, { "text": "Q1" }, { "text": "Q2" }]
              },
              {
                "cells": [{ "text": "100" }, { "text": "120" }]
              },
              {
                "cells": [
                  { "text": "Products", "rowSpan": 2 },
                  { "text": "200" },
                  { "text": "250" }
                ]
              },
              { "cells": [{ "text": "210" }, { "text": "270" }] }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
new TableFrame({
  x: 50,
  y: 50,
  width: 800,
  height: 350,
  rows: [
    {
      cells: [{ text: "Category", rowSpan: 2 }, { text: "Q1" }, { text: "Q2" }],
    },
    {
      cells: [
        // First cell merged from row above
        { text: "100" },
        { text: "120" },
      ],
    },
    {
      cells: [{ text: "Products", rowSpan: 2 }, { text: "200" }, { text: "250" }],
    },
    {
      cells: [{ text: "210" }, { text: "270" }],
    },
  ],
});
```

## Column Widths

Control column widths explicitly:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "table": {
            "x": 50,
            "y": 50,
            "width": 800,
            "columnWidths": [200, 300, 300],
            "rows": [
              { "cells": [{ "text": "ID" }, { "text": "Name" }, { "text": "Email" }] },
              { "cells": [{ "text": "1" }, { "text": "Alice" }, { "text": "alice@example.com" }] }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
new TableFrame({
  x: 50,
  y: 50,
  width: 800,
  columnWidths: [200, 300, 300], // Width for each column
  rows: [
    { cells: [{ text: "ID" }, { text: "Name" }, { text: "Email" }] },
    { cells: [{ text: "1" }, { text: "Alice" }, { text: "alice@example.com" }] },
  ],
});
```

## Borders

Customize cell borders:

```ts
{
    text: "Cell with border",
    borders: {
        top: { width: 2, color: "2E74B5" },
        bottom: { width: 2, color: "2E74B5" },
        left: { width: 2, color: "2E74B5" },
        right: { width: 2, color: "2E74B5" },
    },
}
```

Table-level borders distribute to edge cells:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "table": {
            "x": 50,
            "y": 50,
            "width": 800,
            "rows": [
              { "cells": [{ "text": "A" }, { "text": "B" }] },
              { "cells": [{ "text": "C" }, { "text": "D" }] }
            ],
            "borders": {
              "top": { "width": 2, "color": "2E74B5" },
              "bottom": { "width": 2, "color": "2E74B5" },
              "left": { "width": 2, "color": "2E74B5" },
              "right": { "width": 2, "color": "2E74B5" }
            }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
new TableFrame({
  x: 50,
  y: 50,
  width: 800,
  rows: [{ cells: [{ text: "A" }, { text: "B" }] }, { cells: [{ text: "C" }, { text: "D" }] }],
  borders: {
    top: { width: 2, color: "2E74B5" },
    bottom: { width: 2, color: "2E74B5" },
    left: { width: 2, color: "2E74B5" },
    right: { width: 2, color: "2E74B5" },
  },
});
```

## Table Options

| Property       | Type                | Description              |
| -------------- | ------------------- | ------------------------ |
| `x`            | `number`            | Horizontal position      |
| `y`            | `number`            | Vertical position        |
| `width`        | `number`            | Frame width              |
| `height`       | `number`            | Frame height             |
| `rows`         | `TableRowOptions[]` | Table rows               |
| `columnWidths` | `number[]`          | Width for each column    |
| `borders`      | `object`            | Table-level edge borders |
| `firstRow`     | `boolean`           | Bold first row styling   |
| `bandRow`      | `boolean`           | Alternating row banding  |

### Cell Options

| Property        | Type                            | Description             |
| --------------- | ------------------------------- | ----------------------- |
| `text`          | `string`                        | Plain text content      |
| `children`      | `BaseXmlComponent[]`            | Paragraph objects       |
| `fill`          | `FillOptions`                   | Cell background fill    |
| `borders`       | `object`                        | Cell borders            |
| `columnSpan`    | `number`                        | Horizontal merge count  |
| `rowSpan`       | `number`                        | Vertical merge count    |
| `verticalAlign` | `"TOP" \| "CENTER" \| "BOTTOM"` | Vertical text alignment |
| `margins`       | `{ top, bottom, left, right }`  | Cell padding            |

### Border Options

| Property    | Type                                                | Description  |
| ----------- | --------------------------------------------------- | ------------ |
| `width`     | `number`                                            | Border width |
| `color`     | `string`                                            | Hex color    |
| `dashStyle` | `"solid" \| "dash" \| "dashDot" \| "lgDash" \| ...` | Dash style   |
