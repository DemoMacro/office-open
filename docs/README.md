# office-open

Generate Office Open XML documents (.docx, .pptx) with JS/TS.

## Packages

| Package                                                                               | Description                                  |
| ------------------------------------------------------------------------------------- | -------------------------------------------- |
| [@office-open/docx](docx/)                                                            | Generate .docx files                         |
| [@office-open/pptx](pptx/)                                                            | Generate .pptx files                         |
| [@office-open/core](https://github.com/DemoMacro/office-open/tree/main/packages/core) | Shared XML components, unit converters       |
| [@office-open/xml](https://github.com/DemoMacro/office-open/tree/main/packages/xml)   | XML parsing and serialization                |
| [docx-plus](https://github.com/DemoMacro/office-open/tree/main/packages/docx-plus)    | Compatibility re-export of @office-open/docx |
