title: "Shapes and Text"
description: "Create shapes with formatted text, paragraphs, alignment, lists, and text fitting"
---

Shapes are the primary building blocks of a slide. Each slide's `children` array can contain shape objects with `textBody` for text content.

## Basic Shape

```ts
import { generatePresentation } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": "1.3cm",
            "y": "1.3cm",
            "width": "13.2cm",
            "height": "7.9cm",
            "textBody": { "text": "Simple text" }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: "1.3cm",
        y: "1.3cm",
        width: "13.2cm",
        height: "7.9cm",
        textBody: { text: "Simple text" },
      },
    },
  ],
}
```

::

## Geometry Presets

Use the `geometry` option to set the shape type:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": "2.6cm",
            "y": "2.6cm",
            "width": "5.3cm",
            "height": "4.0cm",
            "geometry": "ellipse",
            "fill": "4472C4",
            "outline": { "color": "2E74B5", "width": 12700 },
            "textBody": { "text": "Ellipse" }
          }
        },
        {
          "shape": {
            "x": "10.6cm",
            "y": "2.6cm",
            "width": "5.3cm",
            "height": "4.0cm",
            "geometry": "triangle",
            "fill": "ED7D31",
            "outline": { "color": "C5504B", "width": 12700 },
            "textBody": { "text": "Triangle" }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
[
  { shape: { geometry: "ellipse", fill: "4472C4", outline: { color: "2E74B5", width: 2 } } },
  { shape: { geometry: "triangle", fill: "ED7D31", outline: { color: "C5504B", width: 2 } } },
];
```

::

Available geometries:

| Geometry      | Shape               |
| ------------- | ------------------- |
| `"rect"`      | Rectangle (default) |
| `"ellipse"`   | Ellipse / circle    |
| `"roundRect"` | Rounded rectangle   |
| `"triangle"`  | Triangle            |
| `"diamond"`   | Diamond             |
| `"star5"`     | Five-pointed star   |
| `"hexagon"`   | Hexagon             |
| `"chevron"`   | Chevron arrow       |
| `"arrow"`     | Arrow               |

Use `outline` to add a border: `{ "color": "2E74B5", "width": 12700 }`. Do NOT use `strokeColor` or `strokeWidth` — these are not valid properties.

## Paragraphs and Text Runs

Use `textBody.children` for full control over text formatting:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": "1.3cm",
            "y": "1.3cm",
            "width": "15.9cm",
            "height": "6.6cm",
            "textBody": {
              "children": [
                {
                  "children": [{ "text": "Bold title", "bold": true, "size": 28, "fill": "2E74B5" }]
                },
                {
                  "children": [
                    { "text": "Normal text with " },
                    { "text": "italic", "italic": true },
                    { "text": " and " },
                    { "text": "colored", "fill": "FF0000", "underline": "SINGLE" }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: "1.3cm",
        y: "1.3cm",
        width: "15.9cm",
        height: "6.6cm",
        textBody: {
          children: [
            {
              paragraph: {
                children: [{ text: "Bold title", bold: true, size: 28, fill: "2E74B5" }],
              },
            },
            {
              paragraph: {
                children: [
                  "Normal text with ",
                  { text: "italic", italic: true },
                  " and ",
                  { text: "colored", fill: "FF0000", underline: "SINGLE" },
                ],
              },
            },
          ],
        },
      },
    },
  ],
}
```

## TextRun Formatting Options

```ts
// Text runs are plain objects with formatting properties:
{
  text: "Formatted text",
  size: 24, // Size in points
  bold: true,
  italic: true,
  underline: "SINGLE", // "SINGLE" | "DOUBLE" | "NONE"
  fill: "4472C4", // Hex color without #
  font: "Calibri", // Font family
}
```

## Alignment

```ts
// Paragraph alignment via the properties option:
{
  paragraph: {
    properties: { alignment: "left" }, // "left" | "center" | "right" | "justify"
    children: [{ text: "Aligned text" }],
  },
}
```

## Bullet Lists

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": "1.3cm",
            "y": "1.3cm",
            "width": "15.9cm",
            "height": "9.3cm",
            "textBody": {
              "children": [
                {
                  "properties": { "bullet": { "type": "char", "char": "●" } },
                  "children": [{ "text": "First item" }]
                },
                {
                  "properties": { "bullet": { "type": "char", "char": "●" } },
                  "children": [{ "text": "Second item" }]
                },
                {
                  "properties": { "bullet": { "type": "autoNum", "format": "arabicPeriod" } },
                  "children": [{ "text": "Numbered item" }]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: "1.3cm",
        y: "1.3cm",
        width: "15.9cm",
        height: "9.3cm",
        textBody: {
          children: [
            {
              paragraph: {
                properties: { bullet: { type: "char", char: "●" } },
                children: ["First item"],
              },
            },
            {
              paragraph: {
                properties: { bullet: { type: "char", char: "●" } },
                children: ["Second item"],
              },
            },
            {
              paragraph: {
                properties: { bullet: { type: "autoNum", format: "arabicPeriod" } },
                children: ["Numbered item"],
              },
            },
          ],
        },
      },
    },
  ],
}
```

## Text Anchor (Vertical Alignment)

Control vertical positioning of text within the shape:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": "1.3cm",
            "y": "1.3cm",
            "width": "13.2cm",
            "height": "7.9cm",
            "textBody": {
              "anchor": "top",
              "text": "Top-aligned text"
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: "1.3cm",
        y: "1.3cm",
        width: "13.2cm",
        height: "7.9cm",
        textBody: {
          anchor: "top", // "top" | "center" | "bottom" | "justify" | "distribute"
          text: "Top-aligned text",
        },
      },
    },
  ],
}
```

## Auto-Fit Text

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": "1.3cm",
            "y": "1.3cm",
            "width": "10.6cm",
            "height": "5.3cm",
            "textBody": {
              "text": "This text will shrink to fit the shape",
              "autoFit": "normal"
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: "1.3cm",
        y: "1.3cm",
        width: "10.6cm",
        height: "5.3cm",
        textBody: {
          text: "This text will shrink to fit the shape",
          autoFit: "normal", // Shrink text to fit
          // or autoFit: "shape" to resize the shape
        },
      },
    },
  ],
}
```

## Rotation and Flip

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": "1.3cm",
            "y": "1.3cm",
            "width": "7.9cm",
            "height": "5.3cm",
            "rotation": 45,
            "flipHorizontal": true,
            "textBody": { "text": "Rotated shape" }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: "1.3cm",
        y: "1.3cm",
        width: "7.9cm",
        height: "5.3cm",
        rotation: 45, // Degrees
        flipHorizontal: true, // Horizontal mirror
        textBody: { text: "Rotated shape" },
      },
    },
  ],
}
```

## Text Columns

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": "1.3cm",
            "y": "1.3cm",
            "width": "21.2cm",
            "height": "9.3cm",
            "textBody": {
              "columns": 2,
              "columnSpacing": 0.5,
              "children": [
                { "children": [{ "text": "Column 1 content..." }] },
                { "children": [{ "text": "Column 2 content..." }] }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: "1.3cm",
        y: "1.3cm",
        width: "21.2cm",
        height: "9.3cm",
        textBody: {
          columns: 2,
          columnSpacing: 0.5,
          children: [
            { paragraph: { children: ["Column 1 content..."] } },
            { paragraph: { children: ["Column 2 content..."] } },
          ],
        },
      },
    },
  ],
}
```
