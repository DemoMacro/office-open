Add clickable hyperlinks to shapes or text runs for web navigation, email links, or internal slide references.

## Hyperlink on a Shape

Make an entire shape clickable:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 400,
            "height": 100,
            "textBody": {
              "children": [
                {
                  "children": [
                    {
                      "text": "Visit our website",
                      "fill": "0563C1",
                      "underline": "SINGLE",
                      "hyperlink": { "url": "https://example.com", "tooltip": "Go to example.com" }
                    }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generatePresentation } from "@office-open/pptx";

generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 400,
            height: 100,
            textBody: {
              children: [
                {
                  children: [
                    {
                      text: "Visit our website",
                      fill: "0563C1",
                      underline: "SINGLE",
                      hyperlink: { url: "https://example.com", tooltip: "Go to example.com" },
                    },
                  ],
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```

::

## Hyperlink on Text Run

Apply a hyperlink to a specific portion of text:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 300,
            "textBody": {
              "children": [
                {
                  "children": [
                    { "text": "For more info, visit " },
                    {
                      "text": "our documentation",
                      "fill": "0563C1",
                      "underline": "SINGLE",
                      "hyperlink": {
                        "url": "https://docs.example.com",
                        "tooltip": "Open documentation"
                      }
                    },
                    { "text": " for details." }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generatePresentation } from "@office-open/pptx";

generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 600,
            height: 300,
            textBody: {
              children: [
                {
                  children: [
                    { text: "For more info, visit " },
                    {
                      text: "our documentation",
                      fill: "0563C1",
                      underline: "SINGLE",
                      hyperlink: {
                        url: "https://docs.example.com",
                        tooltip: "Open documentation",
                      },
                    },
                    { text: " for details." },
                  ],
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```

## Email Link

Use a `mailto:` URL to create an email hyperlink:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 400,
            "height": 100,
            "textBody": {
              "children": [
                {
                  "children": [
                    {
                      "text": "Contact Support",
                      "fill": "0563C1",
                      "underline": "SINGLE",
                      "hyperlink": {
                        "url": "mailto:support@example.com",
                        "tooltip": "Send email to support"
                      }
                    }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generatePresentation } from "@office-open/pptx";

generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 400,
            height: 100,
            textBody: {
              children: [
                {
                  children: [
                    {
                      text: "Contact Support",
                      fill: "0563C1",
                      underline: "SINGLE",
                      hyperlink: {
                        url: "mailto:support@example.com",
                        tooltip: "Send email to support",
                      },
                    },
                  ],
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```

## Tooltip

## Hyperlink Options

| Property  | Type     | Description                    |
| --------- | -------- | ------------------------------ |
| `url`     | `string` | External URL or `mailto:` link |
| `slide`   | `number` | Target slide index (0-based)   |
| `tooltip` | `string` | Tooltip text shown on hover    |

> Note: Use either `url` or `slide`, not both.
