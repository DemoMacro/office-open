Get started with `@office-open/pptx` by creating a simple multi-slide presentation.

## Installation

```bash
npm install @office-open/pptx
```

## Step 1 — Create a Presentation

```ts
import { Presentation, Slide, Shape, Paragraph, TextRun, Packer } from "@office-open/pptx";

const pres = new Presentation({
  title: "My First Presentation",
  creator: "Author",
  slides: [
    {
      children: [
        new Shape({
          x: 50,
          y: 100,
          width: 800,
          height: 350,
          text: "Hello, PowerPoint!",
          paragraphs: [
            new Paragraph({
              properties: { alignment: "CENTER" },
              children: [
                new TextRun({
                  text: "My First Presentation",
                  fontSize: 40,
                  bold: true,
                  fill: "4472C4",
                }),
              ],
            }),
          ],
        }),
      ],
    },
  ],
});
```

## Step 2 — Add Multiple Slides

```ts
import {
  Presentation,
  Slide,
  Shape,
  Paragraph,
  TextRun,
  Picture,
  Packer,
  convertPixelsToEmu,
} from "@office-open/pptx";

const pres = new Presentation({
  slides: [
    // Title slide
    {
      children: [
        new Shape({
          x: 50,
          y: 150,
          width: 800,
          height: 200,
          text: "Quarterly Report",
          paragraphs: [
            new Paragraph({
              properties: { alignment: "CENTER" },
              children: [new TextRun({ text: "Q4 2025 Summary", fontSize: 44, bold: true })],
            }),
            new Paragraph({
              properties: { alignment: "CENTER" },
              children: [
                new TextRun({
                  text: "Prepared by the Finance Team",
                  fontSize: 20,
                  fill: "808080",
                }),
              ],
            }),
          ],
        }),
      ],
    },

    // Content slide
    {
      children: [
        new Shape({
          x: 50,
          y: 50,
          width: 880,
          height: 100,
          text: "Key Highlights",
          paragraphs: [
            new Paragraph({
              children: [new TextRun({ text: "Key Highlights", fontSize: 28, bold: true })],
            }),
          ],
        }),
        new Shape({
          x: 50,
          y: 100,
          width: 880,
          height: 350,
          paragraphs: [
            new Paragraph({
              properties: { bullet: { type: "char", char: "●" } },
              children: [
                new TextRun({
                  text: "Revenue grew by 15% year-over-year",
                  fontSize: 18,
                }),
              ],
            }),
            new Paragraph({
              properties: { bullet: { type: "char", char: "●" } },
              children: [
                new TextRun({
                  text: "Customer satisfaction reached 94%",
                  fontSize: 18,
                }),
              ],
            }),
            new Paragraph({
              properties: { bullet: { type: "char", char: "●" } },
              children: [new TextRun({ text: "Expanded into 3 new markets", fontSize: 18 })],
            }),
          ],
        }),
      ],
    },
  ],
});
```

## Step 3 — Export the File

### Node.js — Write to file

```ts
import fs from "node:fs";

const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("presentation.pptx", buffer);
console.log("Created presentation.pptx");
```

### Browser — Download

```ts
const blob = await Packer.toBlob(pres);
const url = URL.createObjectURL(blob);
const a = document.createElement("a");
a.href = url;
a.download = "presentation.pptx";
a.click();
URL.revokeObjectURL(url);
```

## Step 4 — Run It

Save the code to a file (e.g., `demo.ts`) and run:

```bash
npx tsx demo.ts
```
