---
title: "Transitions"
description: "Add slide transition effects such as fade, push, wipe, and more"
---

Use the `transition` property on slide options to add animated effects when moving between slides.

## Basic Transition

```ts
{
    children: [],
    transition: { type: "fade", speed: "med" },
}
```

## Transition Types

### Fade

```ts
transition: {
    type: "fade";
}
```

### Push

```ts
transition: { type: "push", dir: "r" }
// dir: "l" (left) | "r" (right) | "u" (up) | "d" (down)
```

### Wipe

```ts
transition: { type: "wipe", dir: "d" }
```

### Cover

```ts
transition: { type: "cover", dir: "r" }
```

### Split

```ts
transition: { type: "split", orient: "horz", dir: "out" }
// orient: "horz" | "vert"
// dir: "in" | "out"
```

### Wheel

```ts
transition: { type: "wheel", spokes: 4 }
```

### Dissolve

```ts
transition: { type: "dissolve", speed: "slow" }
```

### Random

```ts
transition: {
    type: "random";
}
```

## Transition Options

| Property           | Type                        | Description                        |
| ------------------ | --------------------------- | ---------------------------------- |
| `type`             | `TransitionType`            | Transition type                    |
| `speed`            | `"slow" \| "med" \| "fast"` | Transition speed                   |
| `dir`              | `string`                    | Direction (depends on type)        |
| `orient`           | `"horz" \| "vert"`          | Orientation (split/blinds/checker) |
| `spokes`           | `number`                    | Number of spokes (wheel)           |
| `advanceOnClick`   | `boolean`                   | Advance on mouse click             |
| `advanceAfterTime` | `number`                    | Auto-advance after milliseconds    |

### Available Types

`"fade"` | `"push"` | `"wipe"` | `"split"` | `"blinds"` | `"checker"` | `"comb"` | `"randomBar"` | `"cover"` | `"pull"` | `"strips"` | `"wheel"` | `"zoom"` | `"circle"` | `"dissolve"` | `"diamond"` | `"newsflash"` | `"plus"` | `"wedge"` | `"random"` | `"cut"`

## Full Example

```ts
import { Presentation, Shape, Packer } from "@office-open/pptx";
import * as fs from "node:fs";

const pres = new Presentation({
    slides: [
        {
            children: [
                new Shape({
                    x: 100,
                    y: 150,
                    width: 500,
                    height: 100,
                    text: "Welcome",
                    fill: "4472C4",
                }),
            ],
            transition: { type: "fade", speed: "med" },
        },
        {
            children: [
                new Shape({
                    x: 100,
                    y: 150,
                    width: 500,
                    height: 100,
                    text: "Content Slide",
                    fill: "ED7D31",
                }),
            ],
            transition: { type: "push", dir: "r", speed: "slow" },
        },
        {
            children: [
                new Shape({
                    x: 100,
                    y: 150,
                    width: 500,
                    height: 100,
                    text: "Thank You",
                    fill: "70AD47",
                }),
            ],
            transition: { type: "wipe", dir: "d" },
        },
    ],
});

const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("My Presentation.pptx", buffer);
```
