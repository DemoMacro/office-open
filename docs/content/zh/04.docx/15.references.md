## 目录

插入一个可由 Word 应用程序更新的目录域：

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        { "paragraph": { "heading": "Heading1", "children": ["Table of Contents"] } },
        { "toc": { "alias": "Table of Contents", "hyperlink": true, "headingStyleRange": "1-3" } }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  sections: [
    {
      children: [
        { paragraph: { heading: "Heading1", children: ["Table of Contents"] } },
        { toc: { alias: "Table of Contents", hyperlink: true, headingStyleRange: "1-3" } },
      ],
    },
  ],
});
```

## 脚注

在页面底部添加脚注：

::api-example{type="docx"}

```json [JSON]
{
  "footnotes": {
    "1": { "children": ["这是第一条脚注。"] },
    "2": { "children": ["这是第二条脚注。"] }
  },
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              "带有脚注的文字",
              { "footnoteReference": 1 },
              " 和另一个",
              { "footnoteReference": 2 },
              "。"
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  footnotes: {
    1: { children: ["这是第一条脚注。"] },
    2: { children: ["这是第二条脚注。"] },
  },
  sections: [
    {
      children: [
        {
          paragraph: {
            children: [
              "带有脚注的文字",
              { footnoteReference: 1 },
              " 和另一个",
              { footnoteReference: 2 },
              "。",
            ],
          },
        },
      ],
    },
  ],
});
```

::

## 尾注

尾注出现在文档或节的末尾：

::api-example{type="docx"}

```json [JSON]
{
  "endnotes": {
    "1": { "children": ["这是一条尾注。"] }
  },
  "sections": [
    {
      "children": [{ "paragraph": { "children": ["带有尾注的文字", { "endnoteReference": 1 }] } }]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  endnotes: {
    1: { children: ["这是一条尾注。"] },
  },
  sections: [
    {
      children: [
        {
          paragraph: {
            children: ["带有尾注的文字", { endnoteReference: 1 }],
          },
        },
      ],
    },
  ],
});
```

::
