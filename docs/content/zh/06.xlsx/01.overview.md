## 快速开始

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Sheet1",
      "rows": [
        { "cells": [{ "value": "Name" }, { "value": "Score" }] },
        { "cells": [{ "value": "Alice" }, { "value": 95 }] },
        { "cells": [{ "value": "Bob" }, { "value": 88 }] }
      ]
    }
  ]
}
```

```ts [Class]
import { Workbook, Packer } from "@office-open/xlsx";

const wb = new Workbook({
  worksheets: [
    {
      name: "Sheet1",
      rows: [
        { cells: [{ value: "Name" }, { value: "Score" }] },
        { cells: [{ value: "Alice" }, { value: 95 }] },
        { cells: [{ value: "Bob" }, { value: 88 }] },
      ],
    },
  ],
});

const buffer = await Packer.toBuffer(wb);
```

## 主要组件

| 组件                    | 说明                                     |
| ----------------------- | ---------------------------------------- |
| `Workbook`              | 根工作簿容器，包含工作表和属性           |
| `Worksheet`             | 单个工作表，包含行、样式和功能           |
| `Row`, `Cell`, `Column` | 行、单元格和列，支持值、样式、高度和宽度 |
| `Styles`                | 基于索引的样式系统（字体、填充、边框）   |
| `SharedStrings`         | 共享字符串表，用于字符串去重             |
| `Table`                 | 数据表格，支持筛选、排序和汇总行         |
| `PivotTable`            | 带聚合和筛选的数据透视表                 |
| `Drawing`               | 工作表中的图片和图表定位                 |
| `Comments`              | 单元格批注，支持作者跟踪和富文本         |
| `Hyperlink`             | 外部和内部超链接                         |
| `MergeCell`             | 跨行跨列合并单元格                       |
| `DataValidation`        | 列表、数字、日期和自定义验证规则         |
| `ConditionalFormat`     | 基于单元格值的格式化规则                 |
| `AutoFilter`            | 列标题筛选下拉框                         |
| `FreezePane`            | 冻结行和列，保持表头可见                 |
| `SheetProtection`       | 密码保护工作表和可编辑范围               |
| `PageSetup`             | 页面大小、边距、方向和打印设置           |
| `HeaderFooter`          | 打印页眉和页脚                           |

## 补丁修改

通过替换 `{{placeholder}}` 占位符来修改已有的 `.xlsx` 模板：

```ts
import { patchWorkbook } from "@office-open/xlsx";

const result = await patchWorkbook({
  outputType: "nodebuffer",
  data: templateBuffer,
  patches: {
    name: { value: "John Doe" },
    amount: { value: 1500 },
    date: { value: "2024-12-31" },
  },
});
```

完整文档请参阅[补丁修改](./18.patch)。

## 解析

将已有的 `.xlsx` 文件解析为 `WorkbookOptions`，用于检查或往返工作流：

```ts
import { parseWorkbook } from "@office-open/xlsx";

const opts = parseWorkbook(buffer);
// opts.worksheets — 工作表数组
// opts.worksheets[0].rows — 包含单元格的行
// opts.title, opts.creator — 核心属性
```

完整文档请参阅[解析](./17.parsing)。

## 导出格式

每个异步方法都有对应的同步版本（例如 `toBufferSync`）。完整列表请参阅[导出](./16.export)页面。

| 方法                        | 返回值                       | 适用场景           |
| --------------------------- | ---------------------------- | ------------------ |
| `Packer.toBuffer(wb)`       | `Buffer`                     | Node.js 文件 I/O   |
| `Packer.toBlob(wb)`         | `Blob`                       | 浏览器下载         |
| `Packer.toBase64String(wb)` | `string`                     | Data URL、API 负载 |
| `Packer.toString(wb)`       | `string`                     | 调试、检查         |
| `Packer.toStream(wb)`       | `ReadableStream<Uint8Array>` | 流式传输大文件     |
