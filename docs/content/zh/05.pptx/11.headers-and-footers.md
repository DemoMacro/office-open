使用 `headerFooter` 在幻灯片中添加一致的元数据 — 日期、幻灯片编号和自定义文本。

## 幻灯片级别的页眉/页脚

在幻灯片定义中直接设置页眉页脚：

::api-example{type="pptx"}

```json [JSON]
{
  "headerFooter": {
    "dateTime": true,
    "footer": "Confidential",
    "slideNumber": true
  },
  "children": [{ "shape": { "x": 1, "y": 1, "width": 8, "height": 4, "text": "Content slide" } }]
}
```

```ts [Class]
import { Presentation, Slide, Shape } from "@office-open/pptx";

const pres = new Presentation({
  slides: [
    {
      headerFooter: {
        dateTime: true,
        footer: "Confidential",
        slideNumber: true,
      },
      children: [
        new Shape({
          x: 100,
          y: 100,
          width: 800,
          height: 400,
          text: "Content slide",
        }),
      ],
    },
  ],
});
```

::

## HeaderFooter 选项

| 属性          | 类型                | 说明                           |
| ------------- | ------------------- | ------------------------------ |
| `dateTime`    | `boolean`           | 显示/隐藏日期时间占位符        |
| `footer`      | `string \| boolean` | 页脚文本，或 `true` 显示占位符 |
| `slideNumber` | `boolean`           | 显示/隐藏幻灯片编号            |
| `header`      | `boolean`           | 显示/隐藏页眉占位符            |

## 单张幻灯片覆盖

在个别幻灯片上覆盖页眉/页脚：

::api-example{type="pptx"}

```json [JSON]
{
  "headerFooter": {
    "footer": "Draft — Not for Distribution",
    "slideNumber": false
  },
  "children": [{ "shape": { "x": 1, "y": 1, "width": 8, "height": 4, "text": "Special slide" } }]
}
```

```ts [Class]
{
    children: [new Shape({ x: 100, y: 100, width: 800, height: 400, text: "Special slide" })],
    headerFooter: {
        footer: "Draft — Not for Distribution",
        slideNumber: false,
    },
}
```

## 完整示例

::api-example{type="pptx"}

```json [JSON]
{
  "title": "Quarterly Report",
  "creator": "Finance Team",
  "headerFooter": {
    "dateTime": true,
    "footer": "Confidential",
    "slideNumber": true
  },
  "children": [
    {
      "shape": {
        "x": 1,
        "y": 1,
        "width": 8,
        "height": 2,
        "paragraphs": [
          {
            "properties": { "alignment": "CENTER" },
            "children": [{ "text": "Q4 2025 Report", "fontSize": 36, "bold": true }]
          }
        ]
      }
    }
  ]
}
```

```ts [Class]
import { Presentation, Slide, Shape, Paragraph, TextRun, Packer } from "@office-open/pptx";
import * as fs from "node:fs";

const pres = new Presentation({
  title: "Quarterly Report",
  creator: "Finance Team",
  slides: [
    {
      headerFooter: {
        dateTime: true,
        footer: "Confidential",
        slideNumber: true,
      },
      children: [
        new Shape({
          x: 100,
          y: 100,
          width: 800,
          height: 200,
          paragraphs: [
            new Paragraph({
              properties: { alignment: "CENTER" },
              children: [new TextRun({ text: "Q4 2025 Report", fontSize: 36, bold: true })],
            }),
          ],
        }),
      ],
    },
  ],
});

const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("report.pptx", buffer);
```
