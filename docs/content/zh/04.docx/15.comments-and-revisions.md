## 批注

使用 `CommentRangeStart`、`CommentRangeEnd` 和 `CommentReference` 为文档内容添加批注：

```ts
import {
    Document,
    Paragraph,
    TextRun,
    Comments,
    Comment,
    CommentRangeStart,
    CommentRangeEnd,
    CommentReference,
} from "@office-open/docx";

const doc = new Document({
    comments: {
        comments: new Comments({
            children: [
                new Comment({
                    id: 0,
                    author: "Author Name",
                    date: new Date(),
                    children: [
                        new Paragraph({ children: [new TextRun("Please review this section.")] }),
                    ],
                }),
            ],
        }),
    },
    sections: [
        {
            children: [
                new Paragraph({
                    children: [
                        new CommentRangeStart(0),
                        new TextRun("This text has a comment attached."),
                        new CommentRangeEnd(0),
                        new CommentReference(0),
                    ],
                }),
            ],
        },
    ],
});
```

## 修订跟踪

跟踪文档中的插入、删除和移动：

```ts
import { Paragraph, InsertedTextRun, DeletedTextRun } from "@office-open/docx";

// 插入
new Paragraph({
    children: [
        new InsertedTextRun({
            id: 0,
            author: "Author",
            date: "2024-01-15T10:30:00Z",
            text: "This text was inserted.",
        }),
    ],
});

// 删除
new Paragraph({
    children: [
        new DeletedTextRun({
            id: 1,
            author: "Author",
            date: "2024-01-15T10:30:00Z",
            text: "This text was deleted.",
        }),
    ],
});
```

## 结构化文档标签（内容控件）

使用 SDT 创建可编辑区域和表单控件：

```ts
import { StructuredDocumentTagRun, StructuredDocumentTagBlock } from "@office-open/docx";

// 行内内容控件
new Paragraph({
    children: [
        new StructuredDocumentTagRun({
            properties: {
                alias: "FirstName",
                tag: "first-name",
            },
            children: [new TextRun("John")],
        }),
    ],
});

// 块级内容控件
new StructuredDocumentTagBlock({
    properties: {
        alias: "Section",
        tag: "custom-section",
    },
    children: [new Paragraph("Editable content inside the control.")],
});
```

SDT 支持多种控件类型，适用于表单、模板和文档自动化。

## 文档保护

使用设置限制文档编辑：

```ts
import { Document } from "@office-open/docx";

const doc = new Document({
    features: {
        updateFields: true,
    },
    settings: {
        protection: {
            type: "readOnly",
            password: "secret",
        },
    },
    sections: [
        // ...
    ],
});
```

保护类型：`"readOnly"`、`"comments"`、`"trackedChanges"`、`"forms"`。

## 修订跟踪选项

使用 `InsertedTextRun` 和 `DeletedTextRun` 时的选项：

| 选项     | 类型     | 说明            |
| -------- | -------- | --------------- |
| `id`     | `number` | 修订 ID         |
| `author` | `string` | 修订作者名称    |
| `date`   | `string` | ISO 8601 时间戳 |

两者也接受所有 `TextRun` 属性（`text`、`bold`、`italics`、`color` 等）。
