`@office-open/pptx` supports embedding images into slides, setting background images, and using images as shape fills.

## Embedding a Picture

```ts
import { Picture } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "picture": {
            "x": 50,
            "y": 50,
            "width": 400,
            "height": 300,
            "data": "<Uint8Array>",
            "type": "png"
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Picture({
      x: 50,
      y: 50,
      width: 400,
      height: 300,
      data: new Uint8Array(fs.readFileSync("photo.png")),
      type: "png",
    }),
  ],
}
```

::

## Image from URL (Browser)

```ts
const response = await fetch("https://example.com/image.png");
const arrayBuffer = await response.arrayBuffer();
const data = new Uint8Array(arrayBuffer);

new Picture({
  x: 50,
  y: 50,
  width: 500,
  height: 300,
  data,
  type: "png",
});
```

## Supported Image Types

| Type | Extension        | `type` value |
| ---- | ---------------- | ------------ |
| JPEG | `.jpg` / `.jpeg` | `"jpg"`      |
| PNG  | `.png`           | `"png"`      |
| GIF  | `.gif`           | `"gif"`      |
| BMP  | `.bmp`           | `"bmp"`      |
| EMF  | `.emf`           | `"emf"`      |
| WMF  | `.wmf`           | `"wmf"`      |

## Slide Background Image

Set a background image for the entire slide:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "background": {
        "fill": { "type": "blip", "data": "<Uint8Array>", "imageType": "png" }
      },
      "children": []
    }
  ]
}
```

```ts [Class]
{
  background: {
    fill: {
      type: "blip",
      data: new Uint8Array(fs.readFileSync("background.png")),
      imageType: "png",
    },
  },
  children: [
    // Your slide content goes on top of the background
  ],
}
```

## Gradient Background

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "background": {
        "fill": {
          "type": "gradient",
          "stops": [
            { "position": 0, "color": "0D47A1" },
            { "position": 100, "color": "42A5F5" }
          ],
          "angle": 45
        }
      },
      "children": []
    }
  ]
}
```

```ts [Class]
{
  background: {
    fill: {
      type: "gradient",
      stops: [
        { position: 0, color: "0D47A1" },
        { position: 100, color: "42A5F5" },
      ],
      angle: 45,
    },
  },
  children: [],
}
```

## Image Fill (Blip Fill) on Shapes

Use an image as a fill for any shape:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 400,
            "geometry": "roundRect",
            "fill": { "type": "blip", "data": "<Uint8Array>", "imageType": "png" },
            "text": "Text over image fill"
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 50,
      y: 50,
      width: 600,
      height: 400,
      geometry: "roundRect",
      fill: {
        type: "blip",
        data: new Uint8Array(fs.readFileSync("texture.png")),
        imageType: "png",
      },
      text: "Text over image fill",
    }),
  ],
}
```

## Picture Options

| Property | Type                                                 | Description     |
| -------- | ---------------------------------------------------- | --------------- |
| `x`      | `number`                                             | Left position   |
| `y`      | `number`                                             | Top position    |
| `width`  | `number`                                             | Width           |
| `height` | `number`                                             | Height          |
| `data`   | `Uint8Array`                                         | Image file data |
| `type`   | `"png" \| "jpg" \| "gif" \| "bmp" \| "emf" \| "wmf"` | Image format    |
| `name`   | `string`                                             | Display name    |
