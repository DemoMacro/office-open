---
title: "Slide Layout"
description: "Configure slide size, background, and speaker notes"
---

Control the dimensions, background, and notes for each slide.

## Slide Dimensions

Set slide size using the `size` property on the `Presentation`:

```ts
import { Presentation, Slide, Shape } from "@office-open/pptx";

const pres = new Presentation({
    size: { width: 1280, height: 720 }, // Custom size in pixels
    slides: [
        {
            children: [
                new Shape({ x: 100, y: 100, width: 800, height: 400, text: "Widescreen content" }),
            ],
        },
    ],
});
```

### Size Presets

Use string shortcuts for common aspect ratios:

```ts
const pres = new Presentation({
    size: "16:9", // or "4:3"
    slides: [],
});
```

| Preset   | Dimensions (EMU)   | Pixels (96 DPI) |
| -------- | ------------------ | --------------- |
| `"16:9"` | 12192000 x 6858000 | 1280 x 720      |
| `"4:3"`  | 9144000 x 6858000  | 960 x 720       |

## Slide Background

### Solid Color

```ts
{
    background: { fill: "1F4E79" },
    children: [],
}
```

### Gradient

```ts
{
    background: {
        fill: {
            type: "gradient",
            stops: [
                { position: 0, color: "0D47A1" },
                { position: 100, color: "1976D2" },
            ],
            angle: 90,
        },
    },
    children: [],
}
```

### Image Background

```ts
import * as fs from "node:fs";

{
    background: {
        fill: {
            type: "image",
            data: new Uint8Array(fs.readFileSync("bg.png")),
            type: "png",
        },
    },
    children: [],
}
```

## Speaker Notes

Add notes visible in presenter mode:

```ts
{
    children: [new Shape({ x: 100, y: 100, width: 800, height: 400, text: "Slide content" })],
    notes: "Remember to emphasize the key metrics in this slide.",
}
```

## Presentation Metadata

```ts
const pres = new Presentation({
    title: "Annual Report 2025",
    creator: "John Doe",
    description: "Yearly financial summary",
    subject: "Finance",
    keywords: "annual, report, finance",
    lastModifiedBy: "Jane Doe",
    revision: 1,
    slides: [],
});
```

### Metadata Properties

| Property         | Type     | Description          |
| ---------------- | -------- | -------------------- |
| `title`          | `string` | Document title       |
| `creator`        | `string` | Author name          |
| `description`    | `string` | Document description |
| `subject`        | `string` | Subject              |
| `keywords`       | `string` | Keywords             |
| `lastModifiedBy` | `string` | Last modifier name   |
| `revision`       | `number` | Revision number      |
