---
title: "Shapes and Text"
description: "Create shapes with formatted text, paragraphs, alignment, lists, and text fitting"
---

Shapes are the primary building blocks of a slide. A `Shape` can contain text through `Paragraph` and `TextRun` objects.

## Basic Shape

```ts
import { Shape } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [{ "shape": { "x": 1, "y": 1, "width": 5, "height": 3, "text": "Simple text" } }]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 5,
      height: 3,
      text: "Simple text",
    }),
  ],
}
```

::

## Geometry Presets

Use the `geometry` option to set the shape type:

```ts
new Shape({ geometry: "rect" }); // Rectangle (default)
new Shape({ geometry: "ellipse" }); // Ellipse / circle
new Shape({ geometry: "roundRect" }); // Rounded rectangle
new Shape({ geometry: "triangle" });
new Shape({ geometry: "diamond" });
new Shape({ geometry: "star5" });
new Shape({ geometry: "hexagon" });
new Shape({ geometry: "chevron" });
new Shape({ geometry: "arrow" });
```

## Paragraphs and Text Runs

Use `paragraphs` for full control over text formatting:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 1,
            "y": 1,
            "width": 6,
            "height": 3,
            "paragraphs": [
              {
                "children": [
                  { "text": "Bold title", "bold": true, "fontSize": 28, "fill": "2E74B5" }
                ]
              },
              {
                "children": [
                  { "text": "Normal text with " },
                  { "text": "italic", "italic": true },
                  { "text": " and " },
                  { "text": "colored", "fill": "FF0000", "underline": "SINGLE" }
                ]
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 6,
      height: 3,
      paragraphs: [
        new Paragraph({
          children: [new TextRun({ text: "Bold title", bold: true, fontSize: 28, fill: "2E74B5" })],
        }),
        new Paragraph({
          children: [
            new TextRun({ text: "Normal text with " }),
            new TextRun({ text: "italic", italic: true }),
            new TextRun({ text: " and " }),
            new TextRun({ text: "colored", fill: "FF0000", underline: "SINGLE" }),
          ],
        }),
      ],
    }),
  ],
}
```

::

## TextRun Formatting Options

```ts
new TextRun({
  text: "Formatted text",
  fontSize: 24, // Size in points
  bold: true,
  italic: true,
  underline: "SINGLE", // "SINGLE" | "DOUBLE" | "NONE"
  fill: "4472C4", // Hex color without #
  font: "Calibri", // Font family
});
```

## Alignment

```ts
new Paragraph({
  properties: { alignment: "LEFT" }, // "LEFT" | "CENTER" | "RIGHT" | "JUSTIFY"
  children: [new TextRun({ text: "Aligned text" })],
});
```

## Bullet Lists

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 1,
            "y": 1,
            "width": 6,
            "height": 4,
            "paragraphs": [
              {
                "properties": { "bullet": { "type": "char", "char": "●" } },
                "children": [{ "text": "First item" }]
              },
              {
                "properties": { "bullet": { "type": "char", "char": "●" } },
                "children": [{ "text": "Second item" }]
              },
              {
                "properties": { "bullet": { "type": "autoNum", "format": "arabicPeriod" } },
                "children": [{ "text": "Numbered item" }]
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 6,
      height: 4,
      paragraphs: [
        new Paragraph({
          properties: { bullet: { type: "char", char: "●" } },
          children: [new TextRun({ text: "First item" })],
        }),
        new Paragraph({
          properties: { bullet: { type: "char", char: "●" } },
          children: [new TextRun({ text: "Second item" })],
        }),
        new Paragraph({
          properties: { bullet: { type: "autoNum", format: "arabicPeriod" } },
          children: [new TextRun({ text: "Numbered item" })],
        }),
      ],
    }),
  ],
}
```

## Text Anchor (Vertical Alignment)

Control vertical positioning of text within the shape:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 1,
            "y": 1,
            "width": 5,
            "height": 3,
            "textAnchor": "TOP",
            "text": "Top-aligned text"
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 5,
      height: 3,
      textAnchor: "TOP", // "TOP" | "CENTER" | "BOTTOM"
      text: "Top-aligned text",
    }),
  ],
}
```

## Auto-Fit Text

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 1,
            "y": 1,
            "width": 4,
            "height": 2,
            "text": "This text will shrink to fit the shape",
            "textAutoFit": "normal"
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 4,
      height: 2,
      text: "This text will shrink to fit the shape",
      textAutoFit: "normal", // Shrink text to fit
      // or textAutoFit: "shape" to resize the shape
    }),
  ],
}
```

## Text Columns

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 1,
            "y": 1,
            "width": 8,
            "height": 4,
            "textColumns": 2,
            "textColumnSpacing": 0.5,
            "paragraphs": [
              { "children": [{ "text": "Column 1 content..." }] },
              { "children": [{ "text": "Column 2 content..." }] }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 1,
      y: 1,
      width: 8,
      height: 4,
      textColumns: 2,
      textColumnSpacing: 0.5,
      paragraphs: [
        new Paragraph({ children: [new TextRun({ text: "Column 1 content..." })] }),
        new Paragraph({ children: [new TextRun({ text: "Column 2 content..." })] }),
      ],
    }),
  ],
}
```
