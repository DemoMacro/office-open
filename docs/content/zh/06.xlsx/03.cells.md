单元格是工作表的基本构建块。每个单元格包含一个值和可选的样式。

## 单元格值

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Sheet1",
      "rows": [{ "cells": [{ "value": "Hello" }, { "value": 42 }, { "value": true }] }]
    }
  ]
}
```

```ts [TypeScript]
import { generateWorkbook } from "@office-open/xlsx";

const buffer = await generateWorkbook({
  worksheets: [
    {
      name: "Sheet1",
      rows: [{ cells: [{ value: "Hello" }, { value: 42 }, { value: true }] }],
    },
  ],
});
```

::

支持的值类型：

| 类型      | 示例                                | 说明             |
| --------- | ----------------------------------- | ---------------- |
| `string`  | `{ value: "Hello" }`                | 共享字符串引用   |
| `number`  | `{ value: 42 }`                     | 内联数值         |
| `boolean` | `{ value: true }`                   | 布尔值 (0/1)     |
| `Date`    | `{ value: new Date("2024-01-01") }` | 序列号           |
| `null`    | `{ value: null }`                   | 空单元格（跳过） |

## 带样式的单元格

直接为单元格应用样式：

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Sheet1",
      "rows": [
        {
          "cells": [
            {
              "value": "Header",
              "style": { "font": { "bold": true }, "fill": { "color": "4472C4" } }
            },
            { "value": 100, "style": { "numFmt": "#,##0" } }
          ]
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  cells: [
    { value: "Header", style: { font: { bold: true }, fill: { color: "4472C4" } } },
    { value: 100, style: { numFmt: "#,##0" } },
  ],
}
```

::

完整样式选项请参阅[样式](/zh/xlsx/styles)。

## 行高

设置自定义行高：

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Sheet1",
      "rows": [{ "cells": [{ "value": "Tall row" }], "height": 30 }]
    }
  ]
}
```

```ts [TypeScript]
{
  cells: [{ value: "Tall row" }],
  height: 30, // 高度（磅）
}
```

::

## 隐藏行

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Sheet1",
      "rows": [
        { "cells": [{ "value": "Visible" }] },
        { "cells": [{ "value": "Hidden" }], "hidden": true }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  cells: [{ value: "Visible" }],
},
{
  cells: [{ value: "Hidden" }],
  hidden: true,
},
```

::

## 数字格式

常用的数字格式模式：

| 模式         | 输入   | 显示       |
| ------------ | ------ | ---------- |
| `#,##0`      | 1234   | 1,234      |
| `#,##0.00`   | 1234.5 | 1,234.50   |
| `0%`         | 0.85   | 85%        |
| `yyyy-mm-dd` | Date   | 2024-01-01 |

## 单元格选项参考

| 选项         | 类型                                          | 说明                                    |
| ------------ | --------------------------------------------- | --------------------------------------- |
| `value`      | `string \| number \| boolean \| Date \| null` | 单元格值（省略时自动分配引用）          |
| `reference`  | `string`                                      | 显式单元格引用，如 `"A1"`               |
| `style`      | `StyleOptions`                                | 单元格样式（见[样式](/zh/xlsx/styles)） |
| `styleIndex` | `number`                                      | 直接样式索引（覆盖 `style`）            |
| `formula`    | `FormulaOptions`                              | 单元格公式；`value` 为缓存结果          |

## 行选项参考

| 选项     | 类型            | 说明       |
| -------- | --------------- | ---------- |
| `cells`  | `CellOptions[]` | 单元格数组 |
| `height` | `number`        | 行高（磅） |
| `hidden` | `boolean`       | 隐藏行     |
