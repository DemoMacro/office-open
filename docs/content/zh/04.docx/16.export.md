---
title: "导出"
description: "将文档导出为 buffer、blob、base64、字符串或流"
---

`Packer` 类将 `Document` 转换为各种输出格式。每个异步方法（`toBuffer`、`toBlob` 等）都有对应的同步版本（`toBufferSync`、`toBlobSync` 等）。

异步方法使用 Web Worker 实现非阻塞 ZIP 压缩；同步方法会阻塞主线程，但避免了 Worker 开销。

## toBuffer

返回 `Buffer`。适用于 Node.js 文件 I/O。

```ts
import { Packer } from "@office-open/docx";
import { writeFileSync } from "node:fs";

// 异步（非阻塞，使用 Web Worker）
const buffer = await Packer.toBuffer(doc);
writeFileSync("output.docx", buffer);

// 同步（阻塞，无 Worker 开销）
const buffer2 = Packer.toBufferSync(doc);
writeFileSync("output-sync.docx", buffer2);
```

## toBlob

返回 `Blob`。适用于浏览器环境。

```ts
// 异步
const blob = await Packer.toBlob(doc);

// 同步
const blob2 = Packer.toBlobSync(doc);

// 触发下载
const url = URL.createObjectURL(blob);
const a = document.createElement("a");
a.href = url;
a.download = "output.docx";
a.click();
URL.revokeObjectURL(url);
```

## toBase64String

返回 Base64 编码的字符串。适用于 Data URL 或 API 载荷。

```ts
// 异步
const base64 = await Packer.toBase64String(doc);

// 同步
const base642 = Packer.toBase64StringSync(doc);

const dataUrl = `data:application/vnd.openxmlformats-officedocument.wordprocessingml.document;base64,${base64}`;
```

## toBytes

返回 `Uint8Array`。适用于需要原始字节而非 Node.js `Buffer` 的场景。

```ts
// 异步
const bytes = await Packer.toBytes(doc);

// 同步
const bytes2 = Packer.toBytesSync(doc);
```

## toArrayBuffer

返回 `ArrayBuffer`。适用于浏览器或跨平台代码。

```ts
// 异步
const ab = await Packer.toArrayBuffer(doc);

// 同步
const ab2 = Packer.toArrayBufferSync(doc);
```

## toString

返回字符串表示。适用于调试。

```ts
// 异步
const str = await Packer.toString(doc);

// 同步
const str2 = Packer.toStringSync(doc);
```

## toStream

返回 Node.js `Readable` 流。适用于流式传输大文档。此方法始终是同步的。

```ts
import { createWriteStream } from "node:fs";

const stream = Packer.toStream(doc);
stream.pipe(createWriteStream("output.docx"));
```

## compile

返回原始 `Zippable` 文件映射，不执行压缩。适用于需要自定义 ZIP 处理的场景。

```ts
const files = Packer.compile(doc);
// files 是 Zippable — 可传递给自定义 zip 函数
```

## 汇总

| 方法             | 异步返回值             | 同步返回值    | 环境    | 适用场景      |
| ---------------- | ---------------------- | ------------- | ------- | ------------- |
| `toBuffer`       | `Promise<Buffer>`      | `Buffer`      | Node.js | 文件 I/O      |
| `toBlob`         | `Promise<Blob>`        | `Blob`        | 浏览器  | 下载          |
| `toBytes`        | `Promise<Uint8Array>`  | `Uint8Array`  | 任意    | 原始字节      |
| `toBase64String` | `Promise<string>`      | `string`      | 任意    | Data URL、API |
| `toString`       | `Promise<string>`      | `string`      | 任意    | 调试          |
| `toArrayBuffer`  | `Promise<ArrayBuffer>` | `ArrayBuffer` | 任意    | 内存处理      |
| `toStream`       | `Readable`             | —             | Node.js | 大文件        |
| `compile`        | —                      | `Zippable`    | 任意    | 自定义 ZIP    |
