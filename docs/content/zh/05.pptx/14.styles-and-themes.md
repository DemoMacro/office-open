使用填充、轮廓和配色方案在演示文稿中应用一致的样式。

## 形状的纯色填充

```ts
import { generatePresentation } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": "1.3cm",
            "y": "1.3cm",
            "width": "13.2cm",
            "height": "5.3cm",
            "fill": { "type": "solid", "color": "2E74B5" },
            "textBody": {
              "text": "Blue shape",
              "children": [
                {
                  "children": [{ "text": "Blue background", "fill": "FFFFFF", "size": 20 }]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: "1.3cm",
            y: "1.3cm",
            width: "13.2cm",
            height: "5.3cm",
            fill: { type: "solid", color: "2E74B5" },
            textBody: {
              children: [
                {
                  paragraph: {
                    children: [{ text: "Blue background", fill: "FFFFFF", size: 20 }],
                  },
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```

::

## 配色方案提示

定义一致的调色板并复用它：

```ts
import { generatePresentation } from "@office-open/pptx";

const colors = {
  primary: "2E74B5",
  secondary: "70AD47",
  accent: "ED7D31",
  dark: "1F3864",
  light: "D6E4F0",
  white: "FFFFFF",
};

generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: "1.3cm",
            y: "1.3cm",
            width: "13.2cm",
            height: "5.3cm",
            fill: { type: "solid", color: colors.primary },
            textBody: {
              children: [
                {
                  paragraph: {
                    children: [{ text: "Title", fill: colors.white, bold: true, size: 24 }],
                  },
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```
