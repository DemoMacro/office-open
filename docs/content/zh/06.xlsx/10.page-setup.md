## 页面设置

配置工作表的打印选项：

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Landscape",
      "rows": [
        { "cells": [{ "value": "Product" }, { "value": "Q1" }, { "value": "Q2" }] },
        { "cells": [{ "value": "Widget A" }, { "value": 100 }, { "value": 200 }] }
      ],
      "pageSetup": {
        "orientation": "landscape",
        "paperSize": 1,
        "fitToWidth": 1,
        "fitToHeight": 0
      }
    }
  ]
}
```

```ts [TypeScript]
import { generateWorkbook } from "@office-open/xlsx";

await generateWorkbook({
  worksheets: [
    {
      name: "Landscape",
      rows: [
        { cells: [{ value: "Product" }, { value: "Q1" }, { value: "Q2" }] },
        { cells: [{ value: "Widget A" }, { value: 100 }, { value: 200 }] },
      ],
      pageSetup: {
        orientation: "landscape",
        paperSize: 1,
        fitToWidth: 1,
        fitToHeight: 0,
      },
    },
  ],
});
```

## 页眉页脚

为打印页面添加页眉和页脚文本。支持特殊代码：`&P`（页码）、`&N`（总页数）：

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Report",
      "rows": [{ "cells": [{ "value": "Data" }] }],
      "headerFooter": {
        "oddHeader": "Sales Report - &P",
        "oddFooter": "Confidential"
      }
    }
  ]
}
```

```ts [TypeScript]
import { generateWorkbook } from "@office-open/xlsx";

await generateWorkbook({
  worksheets: [
    {
      name: "Report",
      rows: [{ cells: [{ value: "Data" }] }],
      headerFooter: {
        oddHeader: "Sales Report - &P",
        oddFooter: "Confidential",
      },
    },
  ],
});
```

## 工作表标签颜色

通过 `tabColor` 设置工作表标签的颜色：

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Blue Tab",
      "tabColor": { "rgb": "FF4472C4" },
      "rows": [{ "cells": [{ "value": "Blue" }] }]
    },
    {
      "name": "Theme Tab",
      "tabColor": { "theme": 3, "tint": 0.3 },
      "rows": [{ "cells": [{ "value": "Theme color" }] }]
    }
  ]
}
```

```ts [TypeScript]
import { generateWorkbook } from "@office-open/xlsx";

await generateWorkbook({
  worksheets: [
    {
      name: "Blue Tab",
      tabColor: { rgb: "FF4472C4" },
      rows: [{ cells: [{ value: "Blue" }] }],
    },
  ],
});
```

## 页面设置选项参考

| 选项                 | 类型      | 说明                                       |
| -------------------- | --------- | ------------------------------------------ |
| `orientation`        | `string`  | `"portrait"`、`"landscape"` 或 `"default"` |
| `paperSize`          | `number`  | 纸张大小编号（1 = Letter, 9 = A4）         |
| `scale`              | `number`  | 缩放百分比                                 |
| `fitToWidth`         | `number`  | 水平页数（0 = 自动）                       |
| `fitToHeight`        | `number`  | 垂直页数（0 = 自动）                       |
| `pageOrder`          | `string`  | `"downThenOver"` 或 `"overThenDown"`       |
| `useFirstPageNumber` | `boolean` | 使用自定义首页页码                         |
| `firstPageNumber`    | `number`  | 首页页码                                   |

## 页眉页脚选项参考

| 选项          | 类型     | 说明       |
| ------------- | -------- | ---------- |
| `oddHeader`   | `string` | 奇数页页眉 |
| `oddFooter`   | `string` | 奇数页页脚 |
| `evenHeader`  | `string` | 偶数页页眉 |
| `evenFooter`  | `string` | 偶数页页脚 |
| `firstHeader` | `string` | 首页页眉   |
| `firstFooter` | `string` | 首页页脚   |
