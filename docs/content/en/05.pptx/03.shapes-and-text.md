---
title: "Shapes and Text"
description: "Create shapes with formatted text, paragraphs, alignment, lists, and text fitting"
---

Shapes are the primary building blocks of a slide. A `Shape` can contain text through `Paragraph` and `TextRun` objects.

## Basic Shape

```ts
import { Shape } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 500,
            "height": 300,
            "textBody": { "text": "Simple text" }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 50,
      y: 50,
      width: 500,
      height: 300,
      textBody: { text: "Simple text" },
    }),
  ],
}
```

::

## Geometry Presets

Use the `geometry` option to set the shape type:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 100,
            "y": 100,
            "width": 200,
            "height": 150,
            "geometry": "ellipse",
            "fill": "4472C4",
            "outline": { "color": "2E74B5", "width": 12700 },
            "textBody": { "text": "Ellipse" }
          }
        },
        {
          "shape": {
            "x": 400,
            "y": 100,
            "width": 200,
            "height": 150,
            "geometry": "triangle",
            "fill": "ED7D31",
            "outline": { "color": "C5504B", "width": 12700 },
            "textBody": { "text": "Triangle" }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
[
  new Shape({ geometry: "ellipse", fill: "4472C4", outline: { color: "2E74B5", width: 2 } }),
  new Shape({ geometry: "triangle", fill: "ED7D31", outline: { color: "C5504B", width: 2 } }),
];
```

::

Available geometries:

| Geometry      | Shape               |
| ------------- | ------------------- |
| `"rect"`      | Rectangle (default) |
| `"ellipse"`   | Ellipse / circle    |
| `"roundRect"` | Rounded rectangle   |
| `"triangle"`  | Triangle            |
| `"diamond"`   | Diamond             |
| `"star5"`     | Five-pointed star   |
| `"hexagon"`   | Hexagon             |
| `"chevron"`   | Chevron arrow       |
| `"arrow"`     | Arrow               |

Use `outline` to add a border: `{ "color": "2E74B5", "width": 12700 }`. Do NOT use `strokeColor` or `strokeWidth` — these are not valid properties.

## Paragraphs and Text Runs

Use `textBody.children` for full control over text formatting:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 250,
            "textBody": {
              "children": [
                {
                  "children": [
                    { "text": "Bold title", "bold": true, "fontSize": 28, "fill": "2E74B5" }
                  ]
                },
                {
                  "children": [
                    { "text": "Normal text with " },
                    { "text": "italic", "italic": true },
                    { "text": " and " },
                    { "text": "colored", "fill": "FF0000", "underline": "SINGLE" }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 50,
      y: 50,
      width: 600,
      height: 250,
      textBody: {
        children: [
          new Paragraph({
            children: [new TextRun({ text: "Bold title", bold: true, fontSize: 28, fill: "2E74B5" })],
          }),
          new Paragraph({
            children: [
              new TextRun({ text: "Normal text with " }),
              new TextRun({ text: "italic", italic: true }),
              new TextRun({ text: " and " }),
              new TextRun({ text: "colored", fill: "FF0000", underline: "SINGLE" }),
            ],
          }),
        ],
      },
    }),
  ],
}
```

::

## TextRun Formatting Options

```ts
new TextRun({
  text: "Formatted text",
  fontSize: 24, // Size in points
  bold: true,
  italic: true,
  underline: "SINGLE", // "SINGLE" | "DOUBLE" | "NONE"
  fill: "4472C4", // Hex color without #
  font: "Calibri", // Font family
});
```

## Alignment

```ts
new Paragraph({
  properties: { alignment: "left" }, // "left" | "center" | "right" | "justify"
  children: [new TextRun({ text: "Aligned text" })],
});
```

## Bullet Lists

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 350,
            "textBody": {
              "children": [
                {
                  "properties": { "bullet": { "type": "char", "char": "●" } },
                  "children": [{ "text": "First item" }]
                },
                {
                  "properties": { "bullet": { "type": "char", "char": "●" } },
                  "children": [{ "text": "Second item" }]
                },
                {
                  "properties": { "bullet": { "type": "autoNum", "format": "arabicPeriod" } },
                  "children": [{ "text": "Numbered item" }]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 50,
      y: 50,
      width: 600,
      height: 350,
      textBody: {
        children: [
          new Paragraph({
            properties: { bullet: { type: "char", char: "●" } },
            children: [new TextRun({ text: "First item" })],
          }),
          new Paragraph({
            properties: { bullet: { type: "char", char: "●" } },
            children: [new TextRun({ text: "Second item" })],
          }),
          new Paragraph({
            properties: { bullet: { type: "autoNum", format: "arabicPeriod" } },
            children: [new TextRun({ text: "Numbered item" })],
          }),
        ],
      },
    }),
  ],
}
```

## Text Anchor (Vertical Alignment)

Control vertical positioning of text within the shape:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 500,
            "height": 300,
            "textBody": {
              "anchor": "top",
              "text": "Top-aligned text"
            }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 50,
      y: 50,
      width: 500,
      height: 300,
      textBody: {
        anchor: "top", // "top" | "center" | "bottom" | "justify" | "distribute"
        text: "Top-aligned text",
      },
    }),
  ],
}
```

## Auto-Fit Text

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 400,
            "height": 200,
            "textBody": {
              "text": "This text will shrink to fit the shape",
              "autoFit": "normal"
            }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 50,
      y: 50,
      width: 400,
      height: 200,
      textBody: {
        text: "This text will shrink to fit the shape",
        autoFit: "normal", // Shrink text to fit
        // or autoFit: "shape" to resize the shape
      },
    }),
  ],
}
```

## Text Columns

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 800,
            "height": 350,
            "textBody": {
              "columns": 2,
              "columnSpacing": 0.5,
              "children": [
                { "children": [{ "text": "Column 1 content..." }] },
                { "children": [{ "text": "Column 2 content..." }] }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 50,
      y: 50,
      width: 800,
      height: 350,
      textBody: {
        columns: 2,
        columnSpacing: 0.5,
        children: [
          new Paragraph({ children: [new TextRun({ text: "Column 1 content..." })] }),
          new Paragraph({ children: [new TextRun({ text: "Column 2 content..." })] }),
        ],
      },
    }),
  ],
}
```
