## 数学公式

Math API 允许使用 OOXML 的 Office 数学标记语言（OMML）构建数学公式。

### 基本公式

```ts
import { Paragraph, Math: MathAPI, MathRun, MathFraction } from "@office-open/docx";

new Paragraph({
  children: [
    new MathAPI({
      children: [
        new MathFraction({
          numerator: [new MathRun("a")],
          denominator: [new MathRun("b")],
        }),
      ],
    }),
  ],
});
```

### 复杂公式

```ts
import {
  Paragraph, Math: MathAPI, MathFraction,
} from "@office-open/docx";

new Paragraph({
  children: [
    new MathAPI({
      children: [
        // x = (a + b) / c
        new MathRun("x"),
        new MathRun("="),
        new MathFraction({
          numerator: [new MathRun("a + b")],
          denominator: [new MathRun("c")],
        }),
      ],
    }),
  ],
});
```

### 数学组件

| 组件                 | 说明                       |
| -------------------- | -------------------------- |
| `Math`               | 根公式容器                 |
| `MathRun`            | 公式中的文本               |
| `MathFraction`       | 分数（分子/分母）          |
| `MathSubScript`      | 下标表示法                 |
| `MathSuperScript`    | 上标表示法                 |
| `MathSubSuperScript` | 上下标组合                 |
| `MathRadical`        | 平方根和 n 次根            |
| `MathSum`            | 求和符号（∑）              |
| `MathIntegral`       | 积分符号（∫）              |
| `MathLimitLower`     | 下极限表示法               |
| `MathLimitUpper`     | 上极限表示法               |
| `MathFunction`       | 命名函数（如 sin、cos）    |
| `MathMatrix`         | 矩阵（行和列）             |
| `MathRoundBrackets`  | 圆括号分组                 |
| `MathCurlyBrackets`  | 花括号分组                 |
| `createMathAccent`   | 重音符号（帽子、波浪号等） |
| `createMathBar`      | 上划线/下划线              |
| `MathEqArr`          | 公式数组（对齐的公式）     |
| `MathBorderBox`      | 表达式边框框               |
| `MathBox`            | 带运算符模拟的框           |
| `MathGroupChr`       | 组字符（大括号、方括号）   |
| `MathPhant`          | 占位符（不可见占位）       |

### MathRun

`MathRun` 接受字符串或选项对象：

```ts
new MathRun("x");
new MathRun({ text: "x" });
```

### MathFraction

分子和分母接受 `MathComponent[]` 数组：

```ts
new MathFraction({
  numerator: [new MathRun("a")],
  denominator: [new MathRun("b")],
  fractionType: "bar", // "bar"（默认） | "skw" | "lin" | "noBar"
});
```

### MathRadical

```ts
new MathRadical({
  children: [new MathRun("x + y")], // 被开方数
  degree: [new MathRun("3")], // 可选：n 次根的次数
});
```

### MathSubScript / MathSuperScript / MathSubSuperScript

```ts
new MathSubScript({
  children: [new MathRun("x")],
  subScript: [new MathRun("i")],
});

new MathSuperScript({
  children: [new MathRun("x")],
  superScript: [new MathRun("2")],
});

new MathSubSuperScript({
  children: [new MathRun("x")],
  subScript: [new MathRun("i")],
  superScript: [new MathRun("2")],
});
```

### MathSum / MathIntegral

```ts
new MathSum({
  children: [new MathRun("x")],
  subScript: [new MathRun("i=0")],
  superScript: [new MathRun("n")],
});

new MathIntegral({
  children: [new MathRun("f(x)")],
  subScript: [new MathRun("a")],
  superScript: [new MathRun("b")],
});
```

### MathLimitLower / MathLimitUpper

```ts
new MathLimitLower({
  children: [new MathRun("lim")],
  limit: [new MathRun("x→0")],
});
```

### MathFunction

```ts
new MathFunction({
  name: [new MathRun("sin")],
  children: [new MathRun("x")],
});
```

### MathMatrix

```ts
new MathMatrix({
  rows: [
    [new MathRun("a"), new MathRun("b")],
    [new MathRun("c"), new MathRun("d")],
  ],
});
```

### createMathAccent

用于重音符号（帽子、波浪号等）的工厂函数：

```ts
import { createMathAccent, MathRun } from "@office-open/docx";

createMathAccent({
  accentCharacter: "^", // "^" 表示帽子，"~" 表示波浪号等
  children: [new MathRun("x")],
});
```

### createMathBar

用于上划线/下划线的工厂函数：

```ts
import { createMathBar, MathRun } from "@office-open/docx";

createMathBar({
  type: "top", // "top" 或 "bot"
  children: [new MathRun("x")],
});
```

### MathEqArr

用于对齐公式的公式数组：

```ts
new MathEqArr({
  rows: [
    [new MathRun("a"), new MathRun("="), new MathRun("b + c")],
    [new MathRun("d"), new MathRun("="), new MathRun("e + f")],
  ],
});
```

### MathBorderBox

在表达式周围绘制边框，可配置边框和删除线：

```ts
new MathBorderBox({
  children: [new MathRun("a")],
});

// 隐藏上边框和下边框
new MathBorderBox({
  children: [new MathRun("b")],
  properties: {
    hideTop: true,
    hideBottom: true,
  },
});
```

#### BorderBox 属性

| 属性                 | 类型      | 说明                 |
| -------------------- | --------- | -------------------- |
| `hideTop`            | `boolean` | 隐藏上边框           |
| `hideBottom`         | `boolean` | 隐藏下边框           |
| `hideLeft`           | `boolean` | 隐藏左边框           |
| `hideRight`          | `boolean` | 隐藏右边框           |
| `strikeHorizontal`   | `boolean` | 水平删除线           |
| `strikeVertical`     | `boolean` | 垂直删除线           |
| `strikeDiagonalUp`   | `boolean` | 从左下到右上的对角线 |
| `strikeDiagonalDown` | `boolean` | 从左上到右下的对角线 |

### MathBox

将内容包裹在框中，可选择模拟运算符行为：

```ts
new MathBox({
  children: [new MathRun("x + y")],
  properties: { opEmu: true },
});
```

### MathGroupChr

在内容上方或下方添加分组字符（大括号、方括号等）：

```ts
new MathGroupChr({
  children: [
    new MathEqArr({
      rows: [[new MathRun("a")], [new MathRun("b")]],
    }),
  ],
  properties: {
    chr: "{", // 分组字符
    pos: "bot", // 位置："top" 或 "bot"
    vertJc: "top", // 垂直对齐
  },
});
```

### MathPhantom

创建用于间距控制的不可见占位符：

```ts
new MathPhant({
  children: [new MathRun("dy")],
  properties: {
    zeroAsc: true, // 零上升
    zeroDesc: true, // 零下降
  },
});
```

## 符号字符

在段落中插入 Wingdings 等符号字体的字符：

::api-example{type="docx"}

```json [JSON]
{
  "children": [
    {
      "paragraph": {
        "children": [
          "箭头：",
          { "symbolRun": { "char": "F021", "symbolfont": "Wingdings" } },
          " 勾选：",
          { "symbolRun": { "char": "F052", "symbolfont": "Wingdings" } }
        ]
      }
    }
  ]
}
```

```ts [Class]
import { Paragraph, TextRun, SymbolRun } from "@office-open/docx";

new Paragraph({
  children: [
    new TextRun("箭头："),
    new SymbolRun({ char: "F021", symbolfont: "Wingdings" }),
    new TextRun(" 勾选："),
    new SymbolRun({ char: "F052", symbolfont: "Wingdings" }),
  ],
});
```

::

### SymbolRun 选项

| 选项         | 类型      | 说明                            |
| ------------ | --------- | ------------------------------- |
| `char`       | `string`  | 十六进制字符代码（如 `"F021"`） |
| `symbolfont` | `string`  | 字体名称（默认 `"Wingdings"`）  |
| `bold`       | `boolean` | 粗体                            |
| `italics`    | `boolean` | 斜体                            |
| `color`      | `string`  | 十六进制颜色代码                |
| `size`       | `number`  | 字号（半磅）                    |

## 拼音标注

拼音标注为东亚文字提供发音指南（假名、拼音等）：

```ts
import { Paragraph, createRuby } from "@office-open/docx";

new Paragraph({
  children: [
    createRuby({
      base: "漢字", // 基础文本
      text: "かんじ", // 拼音（注音）文本
    }),
    createRuby({
      base: "汉字",
      text: "hànzì",
      language: "zh-CN", // 语言标识符
    }),
  ],
});
```

### 拼音标注选项

| 属性           | 类型                            | 说明                         |
| -------------- | ------------------------------- | ---------------------------- |
| `base`         | `string`                        | 要标注的基础文本             |
| `text`         | `string`                        | 拼音标注文本                 |
| `alignment`    | `"left" \| "center" \| "right"` | 对齐方式（默认：`"center"`） |
| `rubyFontSize` | `number`                        | 字号（半点为单位）           |
| `offset`       | `number`                        | 垂直偏移量（半点为单位）     |
| `language`     | `string`                        | 语言标识符（如 `"ja-JP"`）   |
