使用 `ChartFrame` 在幻灯片中嵌入图表。图表通过扁平的选项对象配置 — 直接传入 `type`、`categories` 和 `series`。

## 基本图表

```ts
import { ChartFrame } from "@office-open/pptx";

{
    children: [
        new ChartFrame({
            x: 1,
            y: 1,
            width: 8,
            height: 5,
            type: "column",
            title: "Quarterly Sales",
            categories: ["Q1", "Q2", "Q3", "Q4"],
            series: [
                { name: "Revenue", values: [10, 20, 15, 25] },
                { name: "Expenses", values: [8, 12, 10, 14] },
            ],
            showLegend: true,
            style: 2,
        }),
    ],
}
```

## 图表类型

| `type`      | 说明       |
| ----------- | ---------- |
| `"column"`  | 纵向柱状图 |
| `"bar"`     | 横向柱状图 |
| `"line"`    | 折线图     |
| `"pie"`     | 饼图       |
| `"area"`    | 面积图     |
| `"scatter"` | 散点图     |

## 饼图

```ts
new ChartFrame({
  x: 2,
  y: 1,
  width: 6,
  height: 5,
  type: "pie",
  title: "Market Share",
  categories: ["Chrome", "Safari", "Firefox", "Edge", "Other"],
  series: [{ name: "Browser", values: [65, 18, 3, 5, 9] }],
});
```

## 折线图

```ts
new ChartFrame({
  x: 1,
  y: 1,
  width: 8,
  height: 5,
  type: "line",
  title: "Monthly Revenue",
  categories: ["Jan", "Feb", "Mar", "Apr", "May", "Jun"],
  series: [
    { name: "2024", values: [10, 15, 13, 20, 25, 30] },
    { name: "2025", values: [12, 18, 16, 22, 28, 35] },
  ],
  style: 2,
});
```

## 散点图

```ts
new ChartFrame({
  x: 1,
  y: 1,
  width: 8,
  height: 5,
  type: "scatter",
  series: [
    { name: "Dataset A", values: [2, 4, 1, 5, 3] },
    { name: "Dataset B", values: [8, 6, 9, 3, 7] },
  ],
});
```

## ChartFrame 选项

| 属性         | 类型                                                          | 说明        |
| ------------ | ------------------------------------------------------------- | ----------- |
| `x`          | `number`                                                      | 水平位置    |
| `y`          | `number`                                                      | 垂直位置    |
| `width`      | `number`                                                      | 宽度        |
| `height`     | `number`                                                      | 高度        |
| `type`       | `"column" \| "bar" \| "line" \| "pie" \| "area" \| "scatter"` | 图表类型    |
| `title`      | `string`                                                      | 图表标题    |
| `categories` | `string[]`                                                    | 分类标签    |
| `series`     | `IChartSeriesData[]`                                          | 数据系列    |
| `showLegend` | `boolean`                                                     | 显示图例    |
| `style`      | `number`                                                      | 图表样式 ID |

### 系列数据

| 属性     | 类型       | 说明     |
| -------- | ---------- | -------- |
| `name`   | `string`   | 系列名称 |
| `values` | `number[]` | 数据值   |
