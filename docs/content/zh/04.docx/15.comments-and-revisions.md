## 批注

使用 `CommentRangeStart`、`CommentRangeEnd` 和 `CommentReference` 为文档内容添加批注：

::api-example{type="docx"}

```json [JSON]
{
  "comments": {
    "children": [{ "id": 0, "author": "Author Name", "children": ["请审阅此部分。"] }]
  },
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              { "commentRangeStart": 0 },
              "这段文字附带批注。",
              { "commentRangeEnd": 0 },
              { "commentReference": 0 }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import {
  Document,
  Paragraph,
  TextRun,
  Comment,
  CommentRangeStart,
  CommentRangeEnd,
  CommentReference,
} from "@office-open/docx";

new Document({
  comments: {
    children: [
      new Comment({
        id: 0,
        author: "Author Name",
        date: new Date(),
        children: [new Paragraph({ children: [new TextRun("请审阅此部分。")] })],
      }),
    ],
    }),
  },
  sections: [
    {
      children: [
        new Paragraph({
          children: [
            new CommentRangeStart(0),
            new TextRun("这段文字附带批注。"),
            new CommentRangeEnd(0),
            new CommentReference(0),
          ],
        }),
      ],
    },
  ],
});
```

## 修订跟踪

跟踪文档中的插入、删除和移动：

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              "原始文本，",
              {
                "insertion": {
                  "id": 0,
                  "author": "Author",
                  "date": "2024-01-15T10:30:00Z",
                  "text": "插入的文本"
                }
              },
              "，以及",
              {
                "deletion": {
                  "id": 1,
                  "author": "Author",
                  "date": "2024-01-15T10:30:00Z",
                  "text": "删除的文本"
                }
              },
              "。"
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import { Paragraph, TextRun, InsertedTextRun, DeletedTextRun } from "@office-open/docx";

// 插入
new Paragraph({
  children: [
    new InsertedTextRun({
      id: 0,
      author: "Author",
      date: "2024-01-15T10:30:00Z",
      text: "此文本已被插入。",
    }),
  ],
});

// 删除
new Paragraph({
  children: [
    new DeletedTextRun({
      id: 1,
      author: "Author",
      date: "2024-01-15T10:30:00Z",
      text: "此文本已被删除。",
    }),
  ],
});
```

## 结构化文档标签（内容控件）

使用 SDT 创建可编辑区域和表单控件：

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "sdt": {
            "properties": { "alias": "FullName", "tag": "full-name", "richText": true },
            "children": [{ "paragraph": { "children": ["John Doe"] } }]
          }
        },
        {
          "sdt": {
            "properties": {
              "alias": "Color",
              "tag": "combo-color",
              "comboBox": {
                "items": [
                  { "displayText": "Red", "value": "red" },
                  { "displayText": "Blue", "value": "blue" }
                ],
                "lastValue": "Red"
              }
            },
            "children": [{ "paragraph": { "children": ["Red"] } }]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import {
  StructuredDocumentTagRun,
  StructuredDocumentTagBlock,
  Paragraph,
  TextRun,
} from "@office-open/docx";

// 块级内容控件
new StructuredDocumentTagBlock({
  properties: {
    alias: "FullName",
    tag: "full-name",
    richText: true,
  },
  children: [new Paragraph({ children: [new TextRun("John Doe")] })],
});

// 行内内容控件
new Paragraph({
  children: [
    new StructuredDocumentTagRun({
      properties: {
        alias: "FirstName",
        tag: "first-name",
      },
      children: [new TextRun("John")],
    }),
  ],
});
```

::

SDT 支持多种控件类型，适用于表单、模板和文档自动化。

## 文档保护

使用设置限制文档编辑：

::api-example{type="docx"}

```json [JSON]
{
  "features": {
    "updateFields": true,
    "documentProtection": {
      "edit": "readOnly"
    }
  },
  "sections": [
    {
      "children": [{ "paragraph": { "children": ["受保护的文档内容。"] } }]
    }
  ]
}
```

```ts [Class]
import { Document, Paragraph, TextRun } from "@office-open/docx";

new Document({
  features: {
    updateFields: true,
    documentProtection: {
      edit: "readOnly",
    },
  },
  sections: [
    {
      children: [
        new Paragraph({
          children: [new TextRun("受保护的文档内容。")],
        }),
      ],
    },
  ],
});
```

::

保护类型：`"readOnly"`、`"comments"`、`"trackedChanges"`、`"forms"`。

## 修订跟踪选项

使用 `InsertedTextRun` 和 `DeletedTextRun` 时的选项：

| 选项     | 类型     | 说明            |
| -------- | -------- | --------------- |
| `id`     | `number` | 修订 ID         |
| `author` | `string` | 修订作者名称    |
| `date`   | `string` | ISO 8601 时间戳 |

两者也接受所有 `TextRun` 属性（`text`、`bold`、`italics`、`color` 等）。
