---
title: "导出"
description: "将 .pptx 文件生成为 Buffer、Blob、Base64、字符串或流"
---

`Packer` 将 `Presentation` 转换为各种输出格式。每个异步方法（`toBuffer`、`toBlob` 等）都有对应的同步版本（`toBufferSync`、`toBlobSync` 等）。

异步方法使用 Web Worker 实现非阻塞 ZIP 压缩；同步方法会阻塞主线程，但避免了 Worker 开销。默认压缩策略与 Microsoft Office 一致：XML 条目使用 DEFLATE 级别 1（SuperFast），媒体文件使用 STORED（不压缩）。

## 概览

```ts
import { Packer, Presentation } from "@office-open/pptx";

const pres = new Presentation({
  /* ... */
});
```

| 方法                              | 环境    | 返回值                       |
| --------------------------------- | ------- | ---------------------------- |
| `Packer.toBuffer(pres)`           | Node.js | `Promise<Buffer>`            |
| `Packer.toBufferSync(pres)`       | Node.js | `Buffer`                     |
| `Packer.toBlob(pres)`             | 浏览器  | `Promise<Blob>`              |
| `Packer.toBlobSync(pres)`         | 浏览器  | `Blob`                       |
| `Packer.toBytes(pres)`            | 通用    | `Promise<Uint8Array>`        |
| `Packer.toBytesSync(pres)`        | 通用    | `Uint8Array`                 |
| `Packer.toBase64String(pres)`     | 通用    | `Promise<string>`            |
| `Packer.toBase64StringSync(pres)` | 通用    | `string`                     |
| `Packer.toString(pres)`           | 通用    | `Promise<string>`            |
| `Packer.toStringSync(pres)`       | 通用    | `string`                     |
| `Packer.toStream(pres)`           | 通用    | `ReadableStream<Uint8Array>` |
| `Packer.compile(pres)`            | 任意    | `Zippable`                   |

## toBuffer（Node.js）

返回一个可写入文件的 `Buffer`：

```ts
import { Packer, Presentation } from "@office-open/pptx";
import fs from "node:fs";

const pres = new Presentation({ slides: [] });

// 异步（非阻塞）
const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("output.pptx", buffer);

// 同步（阻塞，无 Worker 开销）
const buffer2 = Packer.toBufferSync(pres);
fs.writeFileSync("output-sync.pptx", buffer2);
```

## toBlob（浏览器）

返回一个用于浏览器下载或上传的 `Blob`：

```ts
// 异步
const blob = await Packer.toBlob(pres);

// 同步
const blob2 = Packer.toBlobSync(pres);

// 下载
const url = URL.createObjectURL(blob);
const a = document.createElement("a");
a.href = url;
a.download = "presentation.pptx";
a.click();
URL.revokeObjectURL(url);
```

## toBase64String

返回 Base64 编码的字符串：

```ts
// 异步
const base64 = await Packer.toBase64String(pres);

// 同步
const base642 = Packer.toBase64StringSync(pres);

// 用于嵌入的 Data URI
const dataUri = `data:application/vnd.openxmlformats-officedocument.presentationml.presentation;base64,${base64}`;
```

## toBytes

返回 `Uint8Array`。适用于需要原始字节而非 Node.js `Buffer` 的场景。

```ts
// 异步
const bytes = await Packer.toBytes(pres);

// 同步
const bytes2 = Packer.toBytesSync(pres);
```

## toString

返回文件内容的原始字符串表示：

```ts
// 异步
const str = await Packer.toString(pres);

// 同步
const str2 = Packer.toStringSync(pres);
```

## toStream

返回 `ReadableStream<Uint8Array>`。适用于流式传输大型演示文稿。同时支持 Node.js 和浏览器。此方法始终是同步的。

```ts
import { createWriteStream } from "node:fs";
import { Readable } from "node:stream";

const stream = Packer.toStream(pres);
Readable.fromWeb(stream).pipe(createWriteStream("output.pptx"));
```

## compile

返回原始 `Zippable` 文件映射，不执行压缩。适用于需要自定义 ZIP 处理的场景。

```ts
const files = Packer.compile(pres);
// files 是 Zippable — 可传递给自定义 zip 函数
```

## 压缩选项

所有输出方法均接受可选的第二个参数用于压缩控制。默认行为与 Microsoft Office 一致：XML 条目使用 DEFLATE 级别 1（SuperFast），媒体文件使用 STORED（不压缩）。Node.js 使用原生 `zlib`；浏览器回退到 fflate（纯 JS）。

| 选项                | 类型     | 说明                                    |
| ------------------- | -------- | --------------------------------------- |
| `compression.xml`   | `number` | XML 条目的 DEFLATE 级别（0–9，默认：1） |
| `compression.media` | `number` | 媒体条目的 DEFLATE 级别（0–9，默认：0） |

```ts
// 默认（MS Office 行为：XML DEFLATE 级别 1，媒体 STORED）
await Packer.toBuffer(pres);

// XML 最大压缩
await Packer.toBuffer(pres, { compression: { xml: 9 } });

// 全部 STORED（不压缩）
await Packer.toBuffer(pres, { compression: { xml: 0 } });
```

## 完整示例 — 保存到文件

```ts
import { Presentation, Shape, Paragraph, TextRun, Packer } from "@office-open/pptx";
import fs from "node:fs";

const pres = new Presentation({
  title: "Export Demo",
  slides: [
    {
      children: [
        new Shape({
          x: 50,
          y: 50,
          width: 800,
          height: 400,
          paragraphs: [
            new Paragraph({
              properties: { alignment: "CENTER" },
              children: [
                new TextRun({
                  text: "Generated Presentation",
                  fontSize: 36,
                  bold: true,
                }),
              ],
            }),
          ],
        }),
      ],
    },
  ],
});

const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("export-demo.pptx", buffer);
console.log("Saved to export-demo.pptx");
```
