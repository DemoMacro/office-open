通过创建一个简单的多幻灯片演示文稿来开始使用 `@office-open/pptx`。

## 安装

```bash
npm install @office-open/pptx
```

## 第一步 — 创建演示文稿

```ts
import { Presentation, Slide, Shape, Paragraph, TextRun, Packer } from "@office-open/pptx";

const pres = new Presentation({
  title: "My First Presentation",
  creator: "Author",
  slides: [
    {
      children: [
        new Shape({
          x: 50,
          y: 50,
          width: 880,
          height: 350,
          textBody: {
            text: "Hello, PowerPoint!",
            children: [
              new Paragraph({
                properties: { alignment: "CENTER" },
                children: [
                  new TextRun({
                    text: "My First Presentation",
                    fontSize: 40,
                    bold: true,
                    fill: "4472C4",
                  }),
                ],
              }),
            ],
          },
        }),
      ],
    },
  ],
});
```

## 第二步 — 添加多张幻灯片

```ts
import {
  Presentation,
  Slide,
  Shape,
  Paragraph,
  TextRun,
  Picture,
  Packer,
  convertPixelsToEmu,
} from "@office-open/pptx";

const pres = new Presentation({
  slides: [
    // 标题幻灯片
    {
      children: [
        new Shape({
          x: 50,
          y: 150,
          width: 880,
          height: 200,
          textBody: {
            text: "Quarterly Report",
            children: [
              new Paragraph({
                properties: { alignment: "CENTER" },
                children: [new TextRun({ text: "Q4 2025 Summary", fontSize: 44, bold: true })],
              }),
              new Paragraph({
                properties: { alignment: "CENTER" },
                children: [
                  new TextRun({
                    text: "Prepared by the Finance Team",
                    fontSize: 20,
                    fill: "808080",
                  }),
                ],
              }),
            ],
          },
        }),
      ],
    },

    // 内容幻灯片
    {
      children: [
        new Shape({
          x: 50,
          y: 50,
          width: 880,
          height: 50,
          textBody: {
            text: "Key Highlights",
            children: [
              new Paragraph({
                children: [new TextRun({ text: "Key Highlights", fontSize: 28, bold: true })],
              }),
            ],
          },
        }),
        new Shape({
          x: 50,
          y: 150,
          width: 880,
          height: 350,
          textBody: {
            children: [
              new Paragraph({
                properties: { bullet: { type: "char", char: "●" } },
                children: [
                  new TextRun({
                    text: "Revenue grew by 15% year-over-year",
                    fontSize: 18,
                  }),
                ],
              }),
              new Paragraph({
                properties: { bullet: { type: "char", char: "●" } },
                children: [
                  new TextRun({
                    text: "Customer satisfaction reached 94%",
                    fontSize: 18,
                  }),
                ],
              }),
              new Paragraph({
                properties: { bullet: { type: "char", char: "●" } },
                children: [new TextRun({ text: "Expanded into 3 new markets", fontSize: 18 })],
              }),
            ],
          },
        }),
      ],
    },
  ],
});
```

## 第三步 — 导出文件

### Node.js — 写入文件

```ts
import fs from "node:fs";

const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("presentation.pptx", buffer);
console.log("Created presentation.pptx");
```

### 浏览器 — 下载

```ts
const blob = await Packer.toBlob(pres);
const url = URL.createObjectURL(blob);
const a = document.createElement("a");
a.href = url;
a.download = "presentation.pptx";
a.click();
URL.revokeObjectURL(url);
```

## 第四步 — 运行

将代码保存到文件（例如 `demo.ts`）并运行：

```bash
npx tsx demo.ts
```
