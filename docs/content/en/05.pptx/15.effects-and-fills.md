---

Enhance shapes with visual effects (shadows, glow, 3D) and fills (solid, gradient, image, transparent).

## Solid Fill

```ts
import { Shape } from "@office-open/pptx";

new Shape({
    x: 1,
    y: 1,
    width: 4,
    height: 3,
    fill: { type: "solid", color: "2E74B5" },
});
```

Short form — pass a color string directly:

```ts
new Shape({
    x: 1,
    y: 1,
    width: 4,
    height: 3,
    fill: "2E74B5",
});
```

## Gradient Fill

### Linear Gradient (with angle)

```ts
new Shape({
    x: 1,
    y: 1,
    width: 6,
    height: 3,
    fill: {
        type: "gradient",
        stops: [
            { position: 0, color: "0D47A1" },
            { position: 100, color: "42A5F5" },
        ],
        angle: 45,
    },
});
```

### Radial Gradient (with path)

```ts
new Shape({
    x: 1,
    y: 1,
    width: 5,
    height: 5,
    geometry: "ellipse",
    fill: {
        type: "gradient",
        stops: [
            { position: 0, color: "FFFFFF" },
            { position: 100, color: "1565C0" },
        ],
        path: "circle",
    },
});
```

### Multi-stop Gradient

```ts
new Shape({
    x: 1,
    y: 1,
    width: 6,
    height: 3,
    fill: {
        type: "gradient",
        stops: [
            { position: 0, color: "E53935" },
            { position: 50, color: "FF9800" },
            { position: 100, color: "FFEB3B" },
        ],
        angle: 90,
    },
});
```

## Image Fill (Blip Fill)

```ts
import * as fs from "node:fs";

new Shape({
    x: 1,
    y: 1,
    width: 6,
    height: 4,
    fill: {
        type: "blip",
        data: new Uint8Array(fs.readFileSync("texture.png")),
        imageType: "png",
    },
});
```

## No Fill (Transparent)

```ts
new Shape({
    x: 1,
    y: 1,
    width: 5,
    height: 3,
    fill: { type: "none" },
    outline: { color: "2E74B5", width: 2 },
});
```

## Outline

```ts
new Shape({
    x: 1,
    y: 1,
    width: 5,
    height: 3,
    fill: { type: "solid", color: "E3F2FD" },
    outline: {
        color: "1565C0",
        width: 2, // Width in EMU
        dashStyle: "solid", // "solid" | "dash" | "dashDot" | "dot" | "lgDash" | "sysDot" | "sysDash"
    },
});
```

## Effects

Use the `effects` property for shadows, glow, reflection, soft edge, and 3D effects:

### Outer Shadow

```ts
new Shape({
    x: 1,
    y: 1,
    width: 200,
    height: 120,
    fill: "ED7D31",
    effects: {
        outerShadow: {
            blur: 50800, // Blur radius (EMU)
            distance: 38100, // Offset distance (EMU)
            direction: 5400000, // Angle (60000ths of a degree)
            color: "000000",
            alpha: 50, // Opacity percentage
        },
    },
});
```

### Inner Shadow

```ts
new Shape({
    x: 1,
    y: 1,
    width: 200,
    height: 120,
    fill: "5B9BD5",
    effects: {
        innerShadow: {
            blur: 40000,
            distance: 30000,
            direction: 5400000,
            color: "000000",
            alpha: 40,
        },
    },
});
```

### Glow

```ts
new Shape({
    x: 1,
    y: 1,
    width: 200,
    height: 120,
    fill: "70AD47",
    effects: {
        glow: {
            radius: 152400, // Glow radius (EMU)
            color: "92D050",
            alpha: 60,
        },
    },
});
```

### Reflection

```ts
new Shape({
    x: 1,
    y: 1,
    width: 200,
    height: 120,
    fill: "FFC000",
    effects: {
        reflection: {
            blurRadius: 6350,
            distance: 38100,
            direction: 5400000,
            startAlpha: 90, // Start opacity percentage
            endAlpha: 0, // End opacity percentage
        },
    },
});
```

### Soft Edge

```ts
new Shape({
    x: 1,
    y: 1,
    width: 200,
    height: 120,
    fill: "BF8F00",
    effects: {
        softEdge: { radius: 50800 },
    },
});
```

### 3D Rotation and Bevel

```ts
new Shape({
    x: 1,
    y: 1,
    width: 200,
    height: 150,
    fill: "FFC000",
    effects: {
        rotation3D: {
            x: 20, // X rotation (degrees)
            y: 30, // Y rotation (degrees)
            z: 10, // Z rotation (degrees)
            perspective: 500, // Perspective value
        },
        bevelTop: { width: 8, height: 8 },
        extrusionH: 50000, // Extrusion height (EMU)
        material: "plastic", // "plastic" | "metal" | "matte" | "flat"
    },
});
```

### Combined Effects

Multiple effects can be applied simultaneously:

```ts
new Shape({
    x: 1,
    y: 1,
    width: 200,
    height: 120,
    fill: "7030A0",
    effects: {
        outerShadow: {
            blur: 40000,
            distance: 30000,
            direction: 2700000,
            color: "000000",
            alpha: 40,
        },
        glow: { radius: 101600, color: "B381E7", alpha: 35 },
    },
});
```

## Effects Options

| Property       | Type               | Description                   |
| -------------- | ------------------ | ----------------------------- |
| `outerShadow`  | `IShadowOptions`   | Outer drop shadow             |
| `innerShadow`  | `IShadowOptions`   | Inner shadow                  |
| `glow`         | `IGlowOptions`     | Glow effect                   |
| `reflection`   | `IReflectionOptions` | Reflection effect           |
| `softEdge`     | `ISoftEdgeOptions` | Soft edge effect              |
| `rotation3D`   | `IRotation3DOptions` | 3D rotation                 |
| `bevelTop`     | `IBevelOptions`    | Top bevel                     |
| `bevelBottom`  | `IBevelOptions`    | Bottom bevel                  |
| `extrusionH`   | `number`           | Extrusion height (EMU)        |
| `material`     | `string`           | Material type                 |
| `lighting`     | `string`           | Lighting type                 |
