---
title: "Charts"
description: "Create bar, line, pie, area, and scatter charts in presentations"
---

Use `ChartFrame` to embed charts in slides. Charts are configured with a flat options object — pass `type`, `categories`, and `series` directly.

## Basic Chart

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "chart": {
            "x": 50,
            "y": 50,
            "width": 800,
            "height": 350,
            "type": "column",
            "title": "Quarterly Sales",
            "categories": ["Q1", "Q2", "Q3", "Q4"],
            "series": [
              { "name": "Revenue", "values": [10, 20, 15, 25] },
              { "name": "Expenses", "values": [8, 12, 10, 14] }
            ],
            "showLegend": true,
            "style": 2
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import { ChartFrame } from "@office-open/pptx";

{
    children: [
        new ChartFrame({
            x: 50,
            y: 50,
            width: 800,
            height: 350,
            type: "column",
            title: "Quarterly Sales",
            categories: ["Q1", "Q2", "Q3", "Q4"],
            series: [
                { name: "Revenue", values: [10, 20, 15, 25] },
                { name: "Expenses", values: [8, 12, 10, 14] },
            ],
            showLegend: true,
            style: 2,
        }),
    ],
}
```

::

## Chart Types

| `type`      | Description        |
| ----------- | ------------------ |
| `"column"`  | Vertical bars      |
| `"bar"`     | Horizontal bars    |
| `"line"`    | Line chart         |
| `"pie"`     | Pie chart          |
| `"area"`    | Area chart         |
| `"scatter"` | Scatter / XY chart |

## Pie Chart

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "chart": {
            "x": 100,
            "y": 100,
            "width": 600,
            "height": 350,
            "type": "pie",
            "title": "Market Share",
            "categories": ["Chrome", "Safari", "Firefox", "Edge", "Other"],
            "series": [{ "name": "Browser", "values": [65, 18, 3, 5, 9] }]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
new ChartFrame({
  x: 100,
  y: 100,
  width: 600,
  height: 350,
  type: "pie",
  title: "Market Share",
  categories: ["Chrome", "Safari", "Firefox", "Edge", "Other"],
  series: [{ name: "Browser", values: [65, 18, 3, 5, 9] }],
});
```

## ChartFrame Options

| Property     | Type                                                          | Description         |
| ------------ | ------------------------------------------------------------- | ------------------- |
| `x`          | `number`                                                      | Horizontal position |
| `y`          | `number`                                                      | Vertical position   |
| `width`      | `number`                                                      | Width               |
| `height`     | `number`                                                      | Height              |
| `type`       | `"column" \| "bar" \| "line" \| "pie" \| "area" \| "scatter"` | Chart type          |
| `title`      | `string`                                                      | Chart title         |
| `categories` | `string[]`                                                    | Category labels     |
| `series`     | `IChartSeriesData[]`                                          | Data series         |
| `showLegend` | `boolean`                                                     | Show legend         |
| `style`      | `number`                                                      | Chart style ID      |

### Series Data

| Property | Type       | Description |
| -------- | ---------- | ----------- |
| `name`   | `string`   | Series name |
| `values` | `number[]` | Data values |
