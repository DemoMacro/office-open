## Merged Cells

Merge a range of cells by specifying the top-left and bottom-right corners using row and column indices (1-based):

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Sheet1",
      "mergeCells": [{ "from": { "row": 1, "col": 1 }, "to": { "row": 1, "col": 4 } }],
      "rows": [
        { "cells": [{ "value": "Merged header across 4 columns" }] },
        { "cells": [{ "value": "Row 2" }, { "value": "Data" }] }
      ]
    }
  ]
}
```

```ts [Class]
{
  name: "Sheet1",
  mergeCells: [
    { from: { row: 1, col: 1 }, to: { row: 1, col: 4 } }, // Merge A1:D1
  ],
  rows: [
    { cells: [{ value: "Merged header across 4 columns" }] },
    { cells: [{ value: "Row 2" }, { value: "Data" }] },
  ],
}
```

::

Multiple merge ranges:

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Sheet1",
      "mergeCells": [
        { "from": { "row": 1, "col": 1 }, "to": { "row": 1, "col": 4 } },
        { "from": { "row": 3, "col": 1 }, "to": { "row": 5, "col": 1 } }
      ],
      "rows": [
        { "cells": [{ "value": "Wide header" }] },
        { "cells": [{ "value": "A" }, { "value": "B" }] },
        { "cells": [{ "value": "Tall cell" }, { "value": "R3" }] },
        { "cells": [{ "value": null }, { "value": "R4" }] },
        { "cells": [{ "value": null }, { "value": "R5" }] }
      ]
    }
  ]
}
```

```ts [Class]
mergeCells: [
  { from: { row: 1, col: 1 }, to: { row: 1, col: 4 } }, // A1:D1
  { from: { row: 3, col: 1 }, to: { row: 5, col: 1 } }, // A3:A5
],
```

::

## Column Widths

Define custom column widths using `min` and `max` to specify the column range:

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Sheet1",
      "columns": [
        { "min": 1, "max": 1, "width": 20 },
        { "min": 2, "max": 4, "width": 12 },
        { "min": 5, "max": 5, "width": 30 }
      ],
      "rows": [
        {
          "cells": [
            { "value": "Wide" },
            { "value": "Med" },
            { "value": "Med" },
            { "value": "Med" },
            { "value": "Extra" }
          ]
        }
      ]
    }
  ]
}
```

```ts [Class]
columns: [
  { min: 1, max: 1, width: 20 },   // Column A
  { min: 2, max: 4, width: 12 },   // Columns B–D
  { min: 5, max: 5, width: 30 },   // Column E
],
```

::

## Hidden Columns & Rows

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Sheet1",
      "columns": [
        { "min": 1, "max": 1, "width": 15 },
        { "min": 2, "max": 2, "width": 15, "hidden": true }
      ],
      "rows": [
        { "cells": [{ "value": "Visible" }, { "value": "Hidden col" }] },
        { "cells": [{ "value": "Row 2" }, { "value": "Hidden col" }], "hidden": true }
      ]
    }
  ]
}
```

```ts [Class]
columns: [
  { min: 1, max: 1, width: 15 },
  { min: 2, max: 2, width: 15, hidden: true }, // Column B hidden
],
rows: [
  { cells: [{ value: "Visible" }, { value: "Hidden col" }] },
  { cells: [{ value: "Row 2" }, { value: "Hidden col" }], hidden: true }, // Row 2 hidden
],
```

::

## MergeCell Options Reference

| Option | Type                           | Description         |
| ------ | ------------------------------ | ------------------- |
| `from` | `{ row: number, col: number }` | Top-left corner     |
| `to`   | `{ row: number, col: number }` | Bottom-right corner |

## Column Options Reference

| Option        | Type      | Description                     |
| ------------- | --------- | ------------------------------- |
| `min`         | `number`  | First column in range (1-based) |
| `max`         | `number`  | Last column in range (1-based)  |
| `width`       | `number`  | Width in character units        |
| `hidden`      | `boolean` | Hide the column                 |
| `customWidth` | `boolean` | Mark as custom width (auto-set) |
