## 基本图片

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Images",
      "rows": [{ "cells": [{ "value": "Label" }, { "value": "Data" }] }],
      "images": [
        {
          "data": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAQAAAC1HAwCAAAAC0lEQVR42mNkYAAAAAYAAjCB0C8AAAAASUVORK5CYII=",
          "type": "png",
          "col": 3,
          "row": 1
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { readFileSync } from "node:fs";
import { generateWorkbook } from "@office-open/xlsx";

const buffer = await generateWorkbook({
  worksheets: [
    {
      name: "Images",
      rows: [{ cells: [{ value: "Label" }, { value: "Data" }] }],
      images: [
        {
          data: readFileSync("./photo.png"),
          type: "png",
          col: 3, // 从 1 开始的列号（D 列）
          row: 1, // 从 1 开始的行号
        },
      ],
    },
  ],
});
```

## 多张图片

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Gallery",
      "rows": [{ "cells": [{ "value": "Photos" }] }],
      "images": [
        {
          "data": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAQAAAC1HAwCAAAAC0lEQVR42mNkYAAAAAYAAjCB0C8AAAAASUVORK5CYII=",
          "type": "png",
          "col": 2,
          "row": 2
        },
        {
          "data": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAQAAAC1HAwCAAAAC0lEQVR42mNkYAAAAAYAAjCB0C8AAAAASUVORK5CYII=",
          "type": "jpg",
          "col": 5,
          "row": 2
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
images: [
  { data: pngData, type: "png", col: 2, row: 2 },
  { data: jpgData, type: "jpg", col: 5, row: 2 },
],
```

::

## 支持的格式

| 格式 | `type` 值 |
| ---- | --------- |
| PNG  | `"png"`   |
| JPEG | `"jpg"`   |

已压缩格式（PNG/JPEG/GIF）在 ZIP 归档中以未压缩方式存储（ZIP_STORED），因为它们本身已是压缩格式；矢量/未压缩格式（EMF/WMF/BMP）使用 DEFLATE。

## 图片选项参考

| 选项   | 类型             | 说明              |
| ------ | ---------------- | ----------------- |
| `data` | `Uint8Array`     | 图片文件数据      |
| `type` | `"png" \| "jpg"` | 图片格式          |
| `col`  | `number`         | 从 1 开始的列位置 |
| `row`  | `number`         | 从 1 开始的行位置 |
