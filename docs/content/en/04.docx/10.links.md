Add navigation to your documents with hyperlinks and bookmarks.

## External Hyperlinks

Link to external URLs:

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              {
                "text": "Visit example.com",
                "hyperlink": { "link": "https://example.com" },
                "underline": { "type": "single" },
                "color": "0563C1"
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  sections: [
    {
      children: [
        {
          paragraph: {
            children: [
              {
                hyperlink: {
                  link: "https://example.com",
                  children: [
                    {
                      text: "Visit example.com",
                      underline: { type: "single" },
                      color: "0563C1",
                    },
                  ],
                },
              },
            ],
          },
        },
      ],
    },
  ],
});
```

::

## Internal Hyperlinks

Link to bookmarks within the same document using `hyperlink` with `anchor`:

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              {
                "text": "Go to Introduction",
                "hyperlink": { "anchor": "section-intro" },
                "color": "0563C1"
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  sections: [
    {
      children: [
        {
          paragraph: {
            children: [
              {
                hyperlink: {
                  anchor: "section-intro",
                  children: [{ text: "Go to Introduction", color: "0563C1" }],
                },
              },
            ],
          },
        },
      ],
    },
  ],
});
```

::

## Bookmarks

Define bookmarks that can be linked to:

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              { "bookmarkStart": { "name": "section-intro", "id": 0 } },
              "Introduction",
              { "bookmarkEnd": 0 }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  sections: [
    {
      children: [
        {
          paragraph: {
            children: [
              { bookmarkStart: { name: "section-intro", id: 0 } },
              "Introduction",
              { bookmarkEnd: 0 },
            ],
          },
        },
      ],
    },
  ],
});
```

::

Use bookmarks with `hyperlink` (using `anchor`) for cross-referencing within a document. The `anchor` property references bookmark names.

## Screen Tips

Add a tooltip that appears on hover:

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              {
                "text": "Example Link",
                "hyperlink": { "link": "https://example.com", "tooltip": "Click to visit Example" },
                "color": "0563C1"
              }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generateDocument } from "@office-open/docx";

await generateDocument({
  sections: [
    {
      children: [
        {
          paragraph: {
            children: [
              {
                hyperlink: {
                  link: "https://example.com",
                  tooltip: "Click to visit Example",
                  children: [{ text: "Example Link", color: "0563C1" }],
                },
              },
            ],
          },
        },
      ],
    },
  ],
});
```
