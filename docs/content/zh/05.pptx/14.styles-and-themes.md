使用填充、轮廓和配色方案在演示文稿中应用一致的样式。

## 形状的纯色填充

```ts
import { Shape, Paragraph, TextRun } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 500,
            "height": 200,
            "fill": { "type": "solid", "color": "2E74B5" },
            "textBody": {
              "text": "Blue shape",
              "children": [
                {
                  "children": [{ "text": "Blue background", "fill": "FFFFFF", "fontSize": 20 }]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new Shape({
      x: 50,
      y: 50,
      width: 500,
      height: 200,
      fill: { type: "solid", color: "2E74B5" },
      textBody: {
        text: "Blue shape",
        children: [
          new Paragraph({
            children: [new TextRun({ text: "Blue background", fill: "FFFFFF", fontSize: 20 })],
          }),
        ],
      },
    }),
  ],
}
```

::

## 渐变填充

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,

          }
        }
      ]
    }
  ]

```

```ts [Class]
{

      width: 600,
      height: 250,
      fill: {
        type: "gradient",

          { position: 50, color: "1976D2" },
          { position: 100, color: "42A5F5" },
        ],
        angle: 45,
      },
    }),
  ],
}
```

## 配色方案提示

定义一致的调色板并复用它：

```ts
const colors = {
  primary: "2E74B5",
  secondary: "70AD47",
  accent: "ED7D31",
  dark: "1F3864",
  light: "D6E4F0",
  white: "FFFFFF",
};

new Shape({
  fill: { type: "solid", color: colors.primary },
  textBody: {
    children: [
      new Paragraph({
        children: [new TextRun({ text: "Title", fill: colors.white, bold: true, fontSize: 24 })],
      }),
    ],
  },
});
```
