形状是幻灯片的主要构建块。一个形状可以通过 `textBody` 属性来包含文本内容。

## 基本形状

```ts
import { generatePresentation } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 500,
            "height": 300,
            "textBody": { "text": "Simple text" }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: 50,
        y: 50,
        width: 500,
        height: 300,
        textBody: { text: "Simple text" },
      },
    },
  ],
}
```

::

## 几何预设

使用 `geometry` 选项设置形状类型：

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 100,
            "y": 100,
            "width": 200,
            "height": 150,
            "geometry": "ellipse",
            "fill": "4472C4",
            "outline": { "color": "2E74B5", "width": 12700 },
            "textBody": { "text": "椭圆" }
          }
        },
        {
          "shape": {
            "x": 400,
            "y": 100,
            "width": 200,
            "height": 150,
            "geometry": "triangle",
            "fill": "ED7D31",
            "outline": { "color": "C5504B", "width": 12700 },
            "textBody": { "text": "三角形" }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
[
  { shape: { geometry: "ellipse", fill: "4472C4", outline: { color: "2E74B5", width: 2 } } },
  { shape: { geometry: "triangle", fill: "ED7D31", outline: { color: "C5504B", width: 2 } } },
];
```

::

可用的几何类型：

| Geometry      | 形状         |
| ------------- | ------------ |
| `"rect"`      | 矩形（默认） |
| `"ellipse"`   | 椭圆/圆形    |
| `"roundRect"` | 圆角矩形     |
| `"triangle"`  | 三角形       |
| `"diamond"`   | 菱形         |
| `"star5"`     | 五角星       |
| `"hexagon"`   | 六边形       |
| `"chevron"`   | V 形箭头     |
| `"arrow"`     | 箭头         |

使用 `outline` 添加边框：`{ "color": "2E74B5", "width": 12700 }`。不要使用 `strokeColor` 或 `strokeWidth`，这些不是有效的属性。

## 段落和文本片段

使用 `textBody.children` 完全控制文本格式：

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 250,
            "textBody": {
              "children": [
                {
                  "children": [{ "text": "Bold title", "bold": true, "size": 28, "fill": "2E74B5" }]
                },
                {
                  "children": [
                    { "text": "Normal text with " },
                    { "text": "italic", "italic": true },
                    { "text": " and " },
                    { "text": "colored", "fill": "FF0000", "underline": "SINGLE" }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: 50,
        y: 50,
        width: 600,
        height: 250,
        textBody: {
          children: [
            {
              children: [{ text: "Bold title", bold: true, size: 28, fill: "2E74B5" }],
            },
            {
              children: [
                "Normal text with ",
                { text: "italic", italic: true },
                " and ",
                { text: "colored", fill: "FF0000", underline: "SINGLE" },
              ],
            },
          ],
        },
      },
    },
  ],
}
```

::

## TextRun 格式选项

```ts
{
  text: "Formatted text",
  size: 24, // 字号（磅）
  bold: true,
  italic: true,
  underline: "SINGLE", // "SINGLE" | "DOUBLE" | "NONE"
  fill: "4472C4", // 十六进制颜色，不带 #
  font: "Calibri", // 字体
}
```

## 对齐方式

```ts
{
  properties: { alignment: "left" }, // "left" | "center" | "right" | "justify"
  children: [{ text: "Aligned text" }],
}
```

```json [JSON]

    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 350,
            "textBody": {
              "children": [
                {
                  "properties": { "bullet": { "type": "char", "char": "●" } },
                  "children": [{ "text": "First item" }]
                },
                {
                  "properties": { "bullet": { "type": "char", "char": "●" } },
                  "children": [{ "text": "Second item" }]
                },
                {
                  "properties": { "bullet": { "type": "autoNum", "format": "arabicPeriod" } },
                  "children": [{ "text": "Numbered item" }]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: 50,
        y: 50,
        width: 600,
        height: 350,
        textBody: {
          children: [
            {
              properties: { bullet: { type: "char", char: "●" } },
              children: [{ text: "First item" }],
            },
            {
              properties: { bullet: { type: "char", char: "●" } },
              children: [{ text: "Second item" }],
            },
            {
              properties: { bullet: { type: "autoNum", format: "arabicPeriod" } },
              children: [{ text: "Numbered item" }],
            },
          ],
        },
```

## 文本锚点（垂直对齐）

控制文本在形状内的垂直位置：

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 500,
            "height": 300,
            "textBody": {
              "anchor": "top",
              "text": "Top-aligned text"
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: 50,
        y: 50,
        width: 500,
        height: 300,
        textBody: {
          anchor: "top", // "top" | "center" | "bottom" | "justify" | "distribute"
          text: "Top-aligned text",
        },
      },
    },
  ],
}
```

## 文本自适应

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 400,
            "height": 200,
            "textBody": {
              "text": "This text will shrink to fit the shape",
              "autoFit": "normal"
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      shape: {
        x: 50,
        y: 50,
        width: 400,
        height: 200,
        textBody: {
          text: "This text will shrink to fit the shape",
          autoFit: "normal", // 自动缩小文本以适应形状
          // 或 autoFit: "shape" 自动调整形状大小
        },
      },
    },
  ],
}
```
