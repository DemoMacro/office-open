---
title: "Lines and Connectors"
description: "Add lines and connector shapes between elements"
---

Use `LineShape` for straight lines and `ConnectorShape` for lines with arrowheads.

## Line Shape

Lines use start/end point coordinates:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "line": {
            "x1": 50,
            "y1": 120,
            "x2": 800,
            "y2": 120,
            "outline": { "color": "4472C4", "width": 2 }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
    children: [
        new LineShape({
            x1: 50,
            y1: 120,
            x2: 800,
            y2: 120,
            outline: { color: "4472C4", width: 2 },
        }),
    ],
}
```

## Connector with Arrowheads

Connectors support arrowhead endpoints:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "connector": {
            "x1": 50,
            "y1": 130,
            "x2": 400,
            "y2": 130,
            "endArrowhead": "triangle",
            "outline": { "color": "4472C4", "width": 2 }
          }
        },
        {
          "connector": {
            "x1": 50,
            "y1": 200,
            "x2": 400,
            "y2": 200,
            "beginArrowhead": "triangle",
            "endArrowhead": "triangle",
            "outline": { "color": "ED7D31", "width": 2 }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new ConnectorShape({
      x1: 50,
      y1: 130,
      x2: 400,
      y2: 130,
      endArrowhead: "triangle",
      outline: { color: "4472C4", width: 2 },
    }),
    new ConnectorShape({
      x1: 50,
      y1: 200,
      x2: 400,
      y2: 200,
      beginArrowhead: "triangle",
      endArrowhead: "triangle",
      outline: { color: "ED7D31", width: 2 },
    }),
  ],
}
```

::

## Arrowhead Types

| Value        | Description     |
| ------------ | --------------- |
| `"triangle"` | Filled triangle |
| `"stealth"`  | Stealth arrow   |
| `"diamond"`  | Diamond shape   |
| `"oval"`     | Oval circle     |
| `"open"`     | Open arrow      |
| `"none"`     | No arrowhead    |

Arrowheads also support size control:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "connector": {
            "x1": 500,
            "y1": 410,
            "x2": 800,
            "y2": 410,
            "endArrowhead": "triangle",
            "arrowheadWidth": "large",
            "arrowheadLength": "large",
            "outline": { "color": "ED7D31", "width": 2 }
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
{
  children: [
    new ConnectorShape({
      x1: 500,
      y1: 410,
      x2: 800,
      y2: 410,
      endArrowhead: "triangle",
      arrowheadWidth: "large",
      arrowheadLength: "large",
      outline: { color: "ED7D31", width: 2 },
    }),
  ],
}
```

## LineShape Options

| Property  | Type             | Description                       |
| --------- | ---------------- | --------------------------------- |
| `x1`      | `number`         | Start X position                  |
| `y1`      | `number`         | Start Y position                  |
| `x2`      | `number`         | End X position                    |
| `y2`      | `number`         | End Y position                    |
| `fill`    | `FillOptions`    | Line fill                         |
| `outline` | `OutlineOptions` | Outline (color, width, dashStyle) |

## ConnectorShape Options

| Property          | Type                             | Description                       |
| ----------------- | -------------------------------- | --------------------------------- |
| `x1`              | `number`                         | Start X position                  |
| `y1`              | `number`                         | Start Y position                  |
| `x2`              | `number`                         | End X position                    |
| `y2`              | `number`                         | End Y position                    |
| `fill`            | `FillOptions`                    | Line fill                         |
| `outline`         | `OutlineOptions`                 | Outline (color, width, dashStyle) |
| `beginArrowhead`  | `ArrowheadType`                  | Arrowhead at start                |
| `endArrowhead`    | `ArrowheadType`                  | Arrowhead at end                  |
| `arrowheadWidth`  | `"small" \| "medium" \| "large"` | Arrowhead width                   |
| `arrowheadLength` | `"small" \| "medium" \| "large"` | Arrowhead length                  |

### OutlineOptions

| Property    | Type                                                                  | Description      |
| ----------- | --------------------------------------------------------------------- | ---------------- |
| `color`     | `string`                                                              | Hex color        |
| `width`     | `number`                                                              | Line width (EMU) |
| `dashStyle` | `"solid" \| "dash" \| "dashDot" \| "lgDash" \| "sysDot" \| "sysDash"` | Dash style       |
