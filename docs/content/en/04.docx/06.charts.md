title: "Charts"

---

Embed charts in documents using `ChartRun`.

## Basic Column Chart

::api-example{type="docx"}

```json [JSON]
{
  "children": [
    {
      "paragraph": {
        "children": [
          {
            "chart": {
              "type": "column",
              "data": {
                "categories": ["Q1", "Q2", "Q3", "Q4"],
                "series": [{ "name": "Sales", "values": [10, 20, 15, 25] }]
              },
              "title": "Quarterly Sales",
              "transformation": { "width": 500, "height": 300 }
            }
          }
        ]
      }
    }
  ]
}
```

```ts [Class]
import { Document, Paragraph, ChartRun } from "@office-open/docx";

{
  children: [
    new Paragraph({
      children: [
        new ChartRun({
          type: "column",
          data: {
            categories: ["Q1", "Q2", "Q3", "Q4"],
            series: [{ name: "Sales", values: [10, 20, 15, 25] }],
          },
          title: "Quarterly Sales",
          transformation: { width: 500, height: 300 },
        }),
      ],
    }),
  ],
}
```

::

## Chart Types

The `type` property selects the chart kind:

| Type    | Value       |
| ------- | ----------- |
| Column  | `"column"`  |
| Bar     | `"bar"`     |
| Line    | `"line"`    |
| Pie     | `"pie"`     |
| Area    | `"area"`    |
| Scatter | `"scatter"` |

## Multi-Series Chart

::api-example{type="docx"}

```json [JSON]
{
  "children": [
    {
      "paragraph": {
        "children": [
          {
            "chart": {
              "type": "column",
              "data": {
                "categories": ["Q1", "Q2", "Q3", "Q4"],
                "series": [
                  { "name": "Product A", "values": [10, 20, 15, 25] },
                  { "name": "Product B", "values": [15, 10, 20, 30] }
                ]
              },
              "transformation": { "width": 500, "height": 300 }
            }
          }
        ]
      }
    }
  ]
}
```

```ts [Class]
new ChartRun({
  type: "column",
  data: {
    categories: ["Q1", "Q2", "Q3", "Q4"],
    series: [
      { name: "Product A", values: [10, 20, 15, 25] },
      { name: "Product B", values: [15, 10, 20, 30] },
    ],
  },
  transformation: { width: 500, height: 300 },
});
```

## Scatter Chart

Scatter charts use `xValues` and `yValues` instead of `categories`:

::api-example{type="docx"}

```json [JSON]
{
  "children": [
    {
      "paragraph": {
        "children": [
          {
            "chart": {
              "type": "scatter",
              "data": {
                "categories": [],
                "series": [
                  {
                    "name": "Dataset 1",
                    "values": [],
                    "xValues": [1, 2, 3, 4, 5],
                    "yValues": [2, 4, 1, 5, 3]
                  }
                ]
              },
              "transformation": { "width": 500, "height": 300 }
            }
          }
        ]
      }
    }
  ]
}
```

```ts [Class]
new ChartRun({
  type: "scatter",
  data: {
    categories: [],
    series: [
      {
        name: "Dataset 1",
        values: [],
        xValues: [1, 2, 3, 4, 5],
        yValues: [2, 4, 1, 5, 3],
      },
    ],
  },
  transformation: { width: 500, height: 300 },
});
```

## ChartRun Options

| Option            | Type                 | Description                     |
| ----------------- | -------------------- | ------------------------------- |
| `type`            | `string`             | Chart type (see table above)    |
| `data`            | `object`             | Chart data (categories, series) |
| `data.categories` | `string[]`           | Category labels                 |
| `data.series`     | `{ name, values }[]` | Series data                     |
| `transformation`  | `{ width, height }`  | Display dimensions              |
| `title`           | `string`             | Chart title                     |
| `showLegend`      | `boolean`            | Show legend (default: `true`)   |
| `style`           | `number`             | Style preset (2–48)             |
| `floating`        | `object`             | Floating positioning options    |
