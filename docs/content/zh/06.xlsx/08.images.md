## 基本图片

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Images",
      "rows": [{ "cells": [{ "value": "Label" }, { "value": "Data" }] }],
      "images": [
        {
          "data": "<Uint8Array>",
          "type": "png",
          "col": 3,
          "row": 1
        }
      ]
    }
  ]
}
```

```ts [Class]
import { readFileSync } from "node:fs";
import { Workbook, Packer } from "@office-open/xlsx";

const wb = new Workbook({
  worksheets: [
    {
      name: "Images",
      rows: [{ cells: [{ value: "Label" }, { value: "Data" }] }],
      images: [
        {
          data: readFileSync("./photo.png"),
          type: "png",
          col: 3, // 从 1 开始的列号（D 列）
          row: 1, // 从 1 开始的行号
        },
      ],
    },
  ],
});
```

::

## 多张图片

::api-example{type="xlsx"}

```json [JSON]
{
  "worksheets": [
    {
      "name": "Gallery",
      "rows": [{ "cells": [{ "value": "Photos" }] }],
      "images": [
        { "data": "<Uint8Array>", "type": "png", "col": 2, "row": 2 },
        { "data": "<Uint8Array>", "type": "jpeg", "col": 5, "row": 2 }
      ]
    }
  ]
}
```

```ts [Class]
images: [
  { data: pngData, type: "png", col: 2, row: 2 },
  { data: jpgData, type: "jpeg", col: 5, row: 2 },
],
```

::

## 支持的格式

| 格式 | `type` 值 |
| ---- | --------- |
| PNG  | `"png"`   |
| JPEG | `"jpeg"`  |

图片在 ZIP 归档中以未压缩方式存储（ZIP_STORED），因为它们本身已经是压缩格式。

## 图片选项参考

| 选项   | 类型                       | 说明              |
| ------ | -------------------------- | ----------------- |
| `data` | `Uint8Array`               | 图片文件数据      |
| `type` | `"png" \| "jpeg" \| "jpg"` | 图片格式          |
| `col`  | `number`                   | 从 1 开始的列位置 |
| `row`  | `number`                   | 从 1 开始的行位置 |
