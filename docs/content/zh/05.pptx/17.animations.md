---
title: "动画效果"
description: "为形状添加进入、强调和退出动画"
---

为形状添加动画效果，控制它们在演示文稿中的出现和行为方式。

## 基本动画

```ts
import { Slide, Shape, Animation, Paragraph, TextRun } from "@office-open/pptx";

new Slide({
    children: [
        new Shape({
            x: 1,
            y: 1,
            width: 6,
            height: 3,
            text: "Animated shape",
            animation: new Animation({
                type: "appear",
            }),
        }),
    ],
});
```

## 进入动画

### 出现

```ts
new Animation({ type: "appear" });
```

### 飞入

```ts
new Animation({
    type: "fly",
    dir: "fromLeft",
    // dir: "fromRight" | "fromTop" | "fromBottom"
});
```

### 淡入

```ts
new Animation({ type: "fadeIn" });
```

### 擦除

```ts
new Animation({
    type: "wipe",
    dir: "right",
    // dir: "left" | "up" | "down"
});
```

### 浮入

```ts
new Animation({
    type: "float",
    dir: "up",
});
```

### 缩放

```ts
new Animation({
    type: "zoom",
    // preset: "in" | "out"
});
```

## 动画选项

```ts
new Animation({
    type: "fly",
    dir: "fromBottom",
    delay: 500, // 延迟（毫秒）
    duration: 1000, // 持续时间（毫秒）
});
```

## 多个形状动画

通过设置延迟使形状依次动画：

```ts
new Slide({
    children: [
        new Shape({
            x: 1,
            y: 1,
            width: 8,
            height: 1,
            text: "First item",
            animation: new Animation({ type: "fadeIn", delay: 0 }),
        }),
        new Shape({
            x: 1,
            y: 2.2,
            width: 8,
            height: 1,
            text: "Second item",
            animation: new Animation({ type: "fadeIn", delay: 500 }),
        }),
        new Shape({
            x: 1,
            y: 3.4,
            width: 8,
            height: 1,
            text: "Third item",
            animation: new Animation({ type: "fadeIn", delay: 1000 }),
        }),
    ],
});
```

## 完整示例

```ts
import {
    Presentation,
    Slide,
    Shape,
    Paragraph,
    TextRun,
    Animation,
    Packer,
} from "@office-open/pptx";
import fs from "node:fs";

const pres = new Presentation({
    slides: [
        new Slide({
            children: [
                new Shape({
                    x: 1,
                    y: 1,
                    width: 8,
                    height: 2,
                    paragraphs: [
                        new Paragraph({
                            alignment: "center",
                            children: [
                                new TextRun({ text: "Animated Title", fontSize: 36, bold: true }),
                            ],
                        }),
                    ],
                    animation: new Animation({ type: "fly", dir: "fromBottom", duration: 800 }),
                }),
                new Shape({
                    x: 2,
                    y: 3.5,
                    width: 6,
                    height: 2,
                    paragraphs: [
                        new Paragraph({
                            alignment: "center",
                            children: [new TextRun({ text: "Content appears next", fontSize: 24 })],
                        }),
                    ],
                    animation: new Animation({ type: "fadeIn", delay: 800, duration: 600 }),
                }),
            ],
        }),
    ],
});

const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("animated.pptx", buffer);
```
