## Quick Start

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [{ "paragraph": { "children": ["Hello World"] } }]
    }
  ]
}
```

```ts [Class]
import { Document, Packer, Paragraph, TextRun } from "@office-open/docx";

const doc = new Document({
  sections: [
    {
      children: [
        new Paragraph({
          children: [new TextRun("Hello World")],
        }),
      ],
    },
  ],
});

const buffer = await Packer.toBuffer(doc);
```

## Main Components

| Component                                                | Description                                            |
| -------------------------------------------------------- | ------------------------------------------------------ |
| `Document`                                               | Root document container with sections and properties   |
| `Paragraph`                                              | Text paragraph with formatting, alignment, and spacing |
| `TextRun`                                                | Inline text with font, size, color, and style          |
| `Table`, `TableRow`, `TableCell`                         | Structured tables with merged cells and borders        |
| `ImageRun`                                               | Embedded images with transformation and positioning    |
| `ChartRun`                                               | Bar, line, pie, area, and scatter charts               |
| `SmartArtRun`                                            | Predefined SmartArt diagrams                           |
| `Header`, `Footer`                                       | Section headers and footers                            |
| `FootnoteReferenceRun`, `EndnoteReferenceRun`            | Footnotes and endnotes                                 |
| `TableOfContents`                                        | Auto-generated table of contents                       |
| `Bookmark`                                               | Internal bookmarks and cross-references                |
| `Math`, `MathRun`                                        | Mathematical equations via Office MathML (OMML)        |
| `Styles`                                                 | Declarative style definitions                          |
| `ExternalHyperlink`, `InternalHyperlink`                 | External and internal hyperlinks                       |
| `Comment`, `CommentReference`                            | Document comments with author and date tracking        |
| `TextBox`                                                | Floating text boxes with content and styling           |
| `CheckBox`                                               | Form checkbox support in documents                     |
| `StructuredDocumentTagRun`, `StructuredDocumentTagBlock` | Structured document tags (SDT) for form-like documents |
| `InsertedTextRun`, `DeletedTextRun`                      | Track changes — insertions and deletions               |
| `Numbering`, `LevelFormat`                               | Numbered and bulleted lists with multiple levels       |
| `SymbolRun`, `PageBreak`, `ColumnBreak`                  | Special characters, page breaks, and column breaks     |
| `SectionProperties`                                      | Page size, margins, orientation, columns, and borders  |

## Patching

Modify existing `.docx` templates by replacing `{{placeholder}}` tokens with new content:

```ts
import { patchDocument, PatchType, TextRun } from "@office-open/docx";

const result = await patchDocument({
  outputType: "nodebuffer",
  data: templateBuffer,
  patches: {
    name: { type: PatchType.PARAGRAPH, children: [new TextRun("John Doe")] },
  },
});
```

See [Patching](./18.patch) for full documentation.

## Export Formats

Every async method has a synchronous counterpart (e.g., `toBufferSync`). See the [Export](./16.export) page for the full list.

| Method                       | Returns                      | Use Case                |
| ---------------------------- | ---------------------------- | ----------------------- |
| `Packer.toBuffer(doc)`       | `Buffer`                     | Node.js file I/O        |
| `Packer.toBlob(doc)`         | `Blob`                       | Browser downloads       |
| `Packer.toBase64String(doc)` | `string`                     | Data URLs, API payloads |
| `Packer.toString(doc)`       | `string`                     | Debugging, inspection   |
| `Packer.toStream(doc)`       | `ReadableStream<Uint8Array>` | Streaming large files   |
