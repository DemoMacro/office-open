---
title: "媒体"
description: "将视频和音频文件嵌入幻灯片，支持封面图片"
---

使用 `video` 和 `audio` 属性将视频和音频内容嵌入演示文稿。

## 视频

```ts
import { generatePresentation } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "video": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 350,
            "data": "<Uint8Array>",
            "type": "mp4"
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  children: [
    {
      video: {
        x: 50,
        y: 50,
        width: 600,
        height: 350,
        data: new Uint8Array(fs.readFileSync("video.mp4")),
        type: "mp4",
      },
    },
  ],
}
```

::

## 带封面图片的视频

封面图片在视频开始播放前显示：

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "video": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 350,
            "data": "<Uint8Array>",
            "type": "mp4",
            "poster": "<Uint8Array>",
            "posterType": "jpg"
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
{
  video: {
    x: 50,
    y: 50,
    width: 600,
    height: 350,
    data: new Uint8Array(fs.readFileSync("video.mp4")),
    type: "mp4",
    poster: new Uint8Array(fs.readFileSync("poster.jpg")),
    posterType: "jpg",
  },
}
```

::

如果未提供 `poster`，将自动使用一个最小透明占位图。

## 支持的格式

### 视频（`type`）

| 值    | 格式 |
| ----- | ---- |
| `mp4` | MP4  |
| `mov` | MOV  |
| `wmv` | WMV  |
| `avi` | AVI  |

### 音频（`type`）

| 值    | 格式 |
| ----- | ---- |
| `mp3` | MP3  |
| `wav` | WAV  |
| `wma` | WMA  |
| `aac` | AAC  |

## 选项

### VideoFrame

| 属性         | 类型                               | 默认值  | 描述             |
| ------------ | ---------------------------------- | ------- | ---------------- |
| `x`          | `number`                           | `0`     | 左侧位置（像素） |
| `y`          | `number`                           | `0`     | 顶部位置（像素） |
| `width`      | `number`                           | `0`     | 宽度（像素）     |
| `height`     | `number`                           | `0`     | 高度（像素）     |
| `data`       | `Uint8Array`                       | -       | 视频文件数据     |
| `type`       | `"mp4" \| "mov" \| "wmv" \| "avi"` | -       | 视频格式         |
| `name`       | `string`                           | auto    | 显示名称         |
| `poster`     | `Uint8Array`                       | auto    | 封面图片数据     |
| `posterType` | `"png" \| "jpg"`                   | `"png"` | 封面图片格式     |
| `animation`  | `AnimationOptions`                 | -       | 播放动画         |

### AudioFrame

| 属性        | 类型                               | 默认值 | 描述             |
| ----------- | ---------------------------------- | ------ | ---------------- |
| `x`         | `number`                           | `0`    | 左侧位置（像素） |
| `y`         | `number`                           | `0`    | 顶部位置（像素） |
| `width`     | `number`                           | `0`    | 宽度（像素）     |
| `height`    | `number`                           | `0`    | 高度（像素）     |
| `data`      | `Uint8Array`                       | -      | 音频文件数据     |
| `type`      | `"mp3" \| "wav" \| "wma" \| "aac"` | -      | 音频格式         |
| `name`      | `string`                           | auto   | 显示名称         |
| `animation` | `AnimationOptions`                 | -      | 播放动画         |

## 提示

- 为视频提供封面图片，以确保在播放前显示视觉预览。
- 音频框架通常较小，放置在幻灯片底部。
- 媒体文件会直接嵌入到 `.pptx` 文件中 — 为保证性能，请控制文件大小。
- 支持的播放取决于演示文稿查看器（PowerPoint、Keynote、Google Slides 等）。
