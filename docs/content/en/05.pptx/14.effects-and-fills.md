title: "Effects and Fills"
description: "Apply shadows, glow, 3D effects, and various fill types to shapes"
---

Enhance shapes with visual effects (shadows, glow, 3D) and fills (solid, gradient, image, transparent).

## Solid Fill

```ts
import { Shape } from "@office-open/pptx";

new Shape({
    x: 1,
    y: 1,
    width: 4,
    height: 3,
    fill: { type: "solid", color: "2E74B5" },
});
```

Short form — pass a color string directly:

```ts
new Shape({
    x: 1,
    y: 1,
    width: 4,
    height: 3,
    fill: "2E74B5",
});
```

## Gradient Fill

### Linear Gradient (with angle)

```ts
new Shape({
    x: 1,
    y: 1,
    width: 6,
    height: 3,
    fill: {
        type: "gradient",
        stops: [
            { position: 0, color: "0D47A1" },
            { position: 100, color: "42A5F5" },
        ],
        angle: 45,
    },
});
```

### Radial Gradient (with path)

```ts
new Shape({
    x: 1,
    y: 1,
    width: 5,
    height: 5,
    geometry: "ellipse",
    fill: {
        type: "gradient",
        stops: [
            { position: 0, color: "FFFFFF" },
            { position: 100, color: "1565C0" },
        ],
        path: "circle",
    },
});
```

### Multi-stop Gradient

```ts
new Shape({
    x: 1,
    y: 1,
    width: 6,
    height: 3,
    fill: {
        type: "gradient",
        stops: [
            { position: 0, color: "E53935" },
            { position: 50, color: "FF9800" },
            { position: 100, color: "FFEB3B" },
        ],
        angle: 90,
    },
});
```

## Image Fill (Blip Fill)

```ts
import fs from "node:fs";

new Shape({
    x: 1,
    y: 1,
    width: 6,
    height: 4,
    fill: {
        type: "blip",
        data: fs.readFileSync("texture.png"),
        imageType: "png",
    },
});
```

## No Fill (Transparent)

```ts
new Shape({
    x: 1,
    y: 1,
    width: 5,
    height: 3,
    fill: { type: "noFill" },
    outline: { color: "2E74B5", width: 2 },
});
```

## Outline

```ts
new Shape({
    x: 1,
    y: 1,
    width: 5,
    height: 3,
    fill: { type: "solid", color: "E3F2FD" },
    outline: {
        color: "1565C0",
        width: 2, // Width in points
        dashStyle: "solid", // "solid" | "dash" | "dashDot" | "dot" | "lgDash"
    },
});
```

## Shadow

```ts
new Shape({
    x: 1,
    y: 1,
    width: 5,
    height: 3,
    fill: { type: "solid", color: "2E74B5" },
    shadow: {
        type: "outer",
        blur: 6, // Blur radius in points
        distance: 3, // Offset distance in points
        direction: 2700000, // Angle in 60000ths of a degree (45° = 2700000)
        color: "000000",
        opacity: 40, // Opacity percentage (0-100)
    },
});
```

## Glow

```ts
new Shape({
    x: 1,
    y: 1,
    width: 5,
    height: 3,
    fill: { type: "solid", color: "2E74B5" },
    effects: {
        glow: {
            color: "42A5F5",
            radius: 10, // Glow radius in points
            opacity: 60,
        },
    },
});
```

## 3D Effects

```ts
new Shape({
    x: 1,
    y: 1,
    width: 5,
    height: 3,
    fill: { type: "solid", color: "2E74B5" },
    effects: {
        threeD: {
            rotation: 30,
            depth: 20,
            color: "1A237E",
        },
    },
});
```

## Combining Effects

```ts
new Shape({
    x: 1,
    y: 1,
    width: 6,
    height: 3,
    fill: {
        type: "gradient",
        stops: [
            { position: 0, color: "1565C0" },
            { position: 100, color: "42A5F5" },
        ],
        angle: 135,
    },
    outline: { color: "0D47A1", width: 1 },
    shadow: {
        type: "outer",
        blur: 8,
        distance: 4,
        direction: 2700000,
        color: "000000",
        opacity: 30,
    },
    text: "Styled Shape",
});
```
