---
title: "Export"
description: "Generate .pptx files as Buffer, Blob, Base64, string, or stream"
---

`Packer` converts a `Presentation` into various output formats. Every async method (`toBuffer`, `toBlob`, etc.) has a synchronous counterpart (`toBufferSync`, `toBlobSync`, etc.).

Async methods use Web Workers for non-blocking ZIP compression; sync methods block the main thread but avoid Worker overhead.

## Overview

```ts
import { Packer, Presentation } from "@office-open/pptx";

const pres = new Presentation({
  /* ... */
});
```

| Method                            | Environment | Returns               |
| --------------------------------- | ----------- | --------------------- |
| `Packer.toBuffer(pres)`           | Node.js     | `Promise<Buffer>`     |
| `Packer.toBufferSync(pres)`       | Node.js     | `Buffer`              |
| `Packer.toBlob(pres)`             | Browser     | `Promise<Blob>`       |
| `Packer.toBlobSync(pres)`         | Browser     | `Blob`                |
| `Packer.toBytes(pres)`            | Both        | `Promise<Uint8Array>` |
| `Packer.toBytesSync(pres)`        | Both        | `Uint8Array`          |
| `Packer.toBase64String(pres)`     | Both        | `Promise<string>`     |
| `Packer.toBase64StringSync(pres)` | Both        | `string`              |
| `Packer.toString(pres)`           | Both        | `Promise<string>`     |
| `Packer.toStringSync(pres)`       | Both        | `string`              |
| `Packer.toStream(pres)`           | Node.js     | `Readable`            |
| `Packer.compile(pres)`            | Any         | `Zippable`            |

## toBuffer (Node.js)

Returns a `Buffer` that can be written to a file:

```ts
import { Packer, Presentation } from "@office-open/pptx";
import fs from "node:fs";

const pres = new Presentation({ slides: [] });

// Async (non-blocking)
const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("output.pptx", buffer);

// Sync (blocking, no Worker overhead)
const buffer2 = Packer.toBufferSync(pres);
fs.writeFileSync("output-sync.pptx", buffer2);
```

## toBlob (Browser)

Returns a `Blob` for browser download or upload:

```ts
// Async
const blob = await Packer.toBlob(pres);

// Sync
const blob2 = Packer.toBlobSync(pres);

// Download
const url = URL.createObjectURL(blob);
const a = document.createElement("a");
a.href = url;
a.download = "presentation.pptx";
a.click();
URL.revokeObjectURL(url);
```

## toBase64String

Returns a Base64-encoded string:

```ts
// Async
const base64 = await Packer.toBase64String(pres);

// Sync
const base642 = Packer.toBase64StringSync(pres);

// Data URI for embedding
const dataUri = `data:application/vnd.openxmlformats-officedocument.presentationml.presentation;base64,${base64}`;
```

## toBytes

Returns a `Uint8Array`. Useful when you need raw bytes without Node.js `Buffer`.

```ts
// Async
const bytes = await Packer.toBytes(pres);

// Sync
const bytes2 = Packer.toBytesSync(pres);
```

## toString

Returns the raw string representation of the file content:

```ts
// Async
const str = await Packer.toString(pres);

// Sync
const str2 = Packer.toStringSync(pres);
```

## toStream (Node.js)

Returns a readable stream for piping. This method is always synchronous.

```ts
import { createWriteStream } from "node:fs";

const stream = Packer.toStream(pres);
stream.pipe(createWriteStream("output.pptx"));
```

## compile

Returns the raw `Zippable` file map without compressing. Useful when you need to customize the ZIP process.

```ts
const files = Packer.compile(pres);
// files is a Zippable — can be passed to custom zip functions
```

## Full Example — Save to File

```ts
import { Presentation, Shape, Paragraph, TextRun, Packer } from "@office-open/pptx";
import fs from "node:fs";

const pres = new Presentation({
  title: "Export Demo",
  slides: [
    {
      children: [
        new Shape({
          x: 50,
          y: 50,
          width: 800,
          height: 400,
          paragraphs: [
            new Paragraph({
              properties: { alignment: "CENTER" },
              children: [
                new TextRun({
                  text: "Generated Presentation",
                  fontSize: 36,
                  bold: true,
                }),
              ],
            }),
          ],
        }),
      ],
    },
  ],
});

const buffer = await Packer.toBuffer(pres);
fs.writeFileSync("export-demo.pptx", buffer);
console.log("Saved to export-demo.pptx");
```
