## 安装

```bash
pnpm add @office-open/pptx
```

## 快速开始

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "paragraphs": [{ "children": ["Hello World"] }]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import { Presentation, Packer, Slide, Shape, Paragraph, TextRun } from "@office-open/pptx";

const presentation = new Presentation({
  slides: [
    {
      children: [
        new Shape({
          paragraphs: [
            new Paragraph({
              children: [new TextRun("Hello World")],
            }),
          ],
        }),
      ],
    },
  ],
});

const buffer = await Packer.toBuffer(presentation);
```

::

## 主要组件

| 组件                             | 说明                                 |
| -------------------------------- | ------------------------------------ |
| `Presentation`                   | 演示文稿根容器，包含幻灯片和属性     |
| `Slide`                          | 单张幻灯片，包含形状、媒体和布局     |
| `Shape`                          | 矩形、圆角矩形和自定义几何形状       |
| `GroupShape`                     | 分组形状，支持集体变换               |
| `Paragraph`, `TextRun`           | 形状内的文本段落和文本块             |
| `Table`, `TableRow`, `TableCell` | 表格，支持合并单元格和样式           |
| `Picture`                        | 幻灯片上的嵌入图片                   |
| `ChartFrame`                     | 柱状图、折线图、饼图、面积图和散点图 |
| `SmartArtFrame`                  | 预定义 SmartArt 图示                 |
| `VideoFrame`, `AudioFrame`       | 嵌入视频和音频                       |
| `LineShape`, `ConnectorShape`    | 元素之间的线条和连接符               |
| `Transition`                     | 幻灯片切换效果                       |
| `Animation`                      | 形状进入、强调和退出动画             |
| `HeaderFooter`                   | 幻灯片页脚，支持日期、编号和文本     |

## 导出格式

| 方法                         | 返回值     | 适用场景           |
| ---------------------------- | ---------- | ------------------ |
| `Packer.toBuffer(doc)`       | `Buffer`   | Node.js 文件 I/O   |
| `Packer.toBlob(doc)`         | `Blob`     | 浏览器下载         |
| `Packer.toBase64String(doc)` | `string`   | Data URL、API 载荷 |
| `Packer.toString(doc)`       | `string`   | 调试、检查         |
| `Packer.toStream(doc)`       | `Readable` | 流式传输大文件     |

## 解析

使用 `parsePresentation` 解析现有 `.pptx` 文件，返回 `PresentationOptions`：

```ts
import { parsePresentation } from "@office-open/pptx";

const opts = parsePresentation(buffer);
// opts.slides![0].children — 形状、表格、图表等
// opts.slides![0].background — 幻灯片背景
// opts.slides![0].transition — 切换效果
// opts.slides![0].notes — 演讲者备注
// opts.title, opts.creator — 核心属性
// opts.size — 幻灯片尺寸（"16:9"、"4:3" 或自定义）
```

详见 [解析](./20.parsing)。

## 幻灯片尺寸

使用 `size` 选项设置幻灯片尺寸。接受 `"16:9"`、`"4:3"` 或自定义 `{ width, height }` 对象（单位为像素）。

```ts
const presentation = new Presentation({
  size: "16:9",
  slides: [
    // ...
  ],
});
```

| 尺寸                | 说明                  |
| ------------------- | --------------------- |
| `"16:9"`            | 13.33" × 7.5"（宽屏） |
| `"4:3"`             | 10" × 7.5"（标准）    |
| `{ width, height }` | 自定义尺寸（像素）    |
