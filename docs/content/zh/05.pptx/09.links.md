---
title: "链接"
description: "为形状和文本添加超链接以实现交互式导航"
---

为形状或文本片段添加可点击的超链接，用于网页导航、邮件链接或幻灯片内部引用。

## 形状上的超链接

使整个形状可点击：

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 400,
            "height": 100,
            "textBody": {
              "children": [
                {
                  "children": [
                    {
                      "text": "Visit our website",
                      "fill": "0563C1",
                      "underline": "SINGLE",
                      "hyperlink": { "url": "https://example.com", "tooltip": "Go to example.com" }
                    }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generatePresentation } from "@office-open/pptx";

generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 400,
            height: 100,
            textBody: {
              children: [
                {
                  children: [
                    {
                      text: "Visit our website",
                      fill: "0563C1",
                      underline: "SINGLE",
                      hyperlink: { url: "https://example.com", tooltip: "Go to example.com" },
                    },
                  ],
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```

::

## 文本片段上的超链接

为特定文本部分应用超链接：

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 300,
            "textBody": {
              "children": [
                {
                  "children": [
                    { "text": "For more info, visit " },
                    {
                      "text": "our documentation",
                      "fill": "0563C1",
                      "underline": "SINGLE",
                      "hyperlink": {
                        "url": "https://docs.example.com",
                        "tooltip": "Open documentation"
                      }
                    },
                    { "text": " for details." }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generatePresentation } from "@office-open/pptx";

generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 600,
            height: 300,
            textBody: {
              children: [
                {
                  children: [
                    { text: "For more info, visit " },
                    {
                      text: "our documentation",
                      fill: "0563C1",
                      underline: "SINGLE",
                      hyperlink: {
                        url: "https://docs.example.com",
                        tooltip: "Open documentation",
                      },
                    },
                    { text: " for details." },
                  ],
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```

## 邮件链接

使用 `mailto:` URL 创建邮件超链接：

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 400,
            "height": 100,
            "textBody": {
              "children": [
                {
                  "children": [
                    {
                      "text": "Contact Support",
                      "fill": "0563C1",
                      "underline": "SINGLE",
                      "hyperlink": {
                        "url": "mailto:support@example.com",
                        "tooltip": "Send email to support"
                      }
                    }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generatePresentation } from "@office-open/pptx";

generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 400,
            height: 100,
            textBody: {
              children: [
                {
                  children: [
                    {
                      text: "Contact Support",
                      fill: "0563C1",
                      underline: "SINGLE",
                      hyperlink: {
                        url: "mailto:support@example.com",
                        tooltip: "Send email to support",
                      },
                    },
                  ],
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```

::

## 超链接选项

| 属性      | 类型     | 说明                        |
| --------- | -------- | --------------------------- |
| `url`     | `string` | 外部 URL 或 `mailto:` 链接  |
| `slide`   | `number` | 目标幻灯片索引（从 0 开始） |
| `tooltip` | `string` | 悬停时显示的提示文字        |

> 注意：使用 `url` 或 `slide` 其中之一，不要同时使用两者。
