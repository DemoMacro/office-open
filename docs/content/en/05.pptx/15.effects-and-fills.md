Enhance shapes with visual effects (shadows, glow, 3D) and fills (solid, gradient, image, transparent).

## Solid Fill

```ts
import { generatePresentation } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 400,
            "height": 250,
            "fill": { "type": "solid", "color": "2E74B5" }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 400,
            height: 250,
            fill: { type: "solid", color: "2E74B5" },
          },
        },
      ],
    },
  ],
});
```

::

## Image Fill (Blip Fill)

The `data` field accepts raw bytes (`Uint8Array`, `ArrayBuffer`, `Buffer`) or a base64 [data URL](https://developer.mozilla.org/en-US/docs/Web/HTTP/Basics_of_HTTP/Data_URIs) such as `data:image/png;base64,...`.

```ts
import * as fs from "node:fs";
import { generatePresentation } from "@office-open/pptx";
```

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 600,
            "height": 350,
            "fill": {
              "type": "blip",
              "data": "<Uint8Array>",
              "imageType": "png"
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 600,
            height: 350,
            fill: {
              type: "blip",
              data: new Uint8Array(fs.readFileSync("texture.png")),
              imageType: "png",
            },
          },
        },
      ],
    },
  ],
});
```

## Effects

Use the `effects` property for shadows, glow, reflection, soft edge, and 3D effects:

### Outer Shadow

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 200,
            "height": 120,
            "fill": "ED7D31",
            "effects": {
              "outerShadow": {
                "blur": 50800,
                "distance": 38100,
                "direction": 5400000,
                "color": "000000",
                "alpha": 50
              }
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 200,
            height: 120,
            fill: "ED7D31",
            effects: {
              outerShadow: {
                blur: 50800, // Blur radius (EMU)
                distance: 38100, // Offset distance (EMU)
                direction: 5400000, // Angle (60000ths of a degree)
                color: "000000",
                alpha: 50, // Opacity percentage
              },
            },
          },
        },
      ],
    },
  ],
});
```

## Effects Options

| Property      | Type                | Description            |
| ------------- | ------------------- | ---------------------- |
| `outerShadow` | `ShadowOptions`     | Outer drop shadow      |
| `innerShadow` | `ShadowOptions`     | Inner shadow           |
| `glow`        | `GlowOptions`       | Glow effect            |
| `reflection`  | `ReflectionOptions` | Reflection effect      |
| `softEdge`    | `SoftEdgeOptions`   | Soft edge effect       |
| `rotation3D`  | `Rotation3DOptions` | 3D rotation            |
| `bevelTop`    | `PPTXBevelOptions`  | Top bevel              |
| `bevelBottom` | `PPTXBevelOptions`  | Bottom bevel           |
| `extrusionH`  | `number`            | Extrusion height (EMU) |
| `material`    | `string`            | Material type          |
| `lighting`    | `string`            | Lighting type          |
