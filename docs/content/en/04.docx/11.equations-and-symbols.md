## Math Formulas

The Math API lets you build mathematical equations using OOXML's Office Math Markup Language (OMML).

### Basic Equation

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              { "math": { "children": ["E", " = ", "mc", { "text": "2", "superScript": true }] } }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import { Paragraph, Math: MathAPI, MathRun, MathFraction } from "@office-open/docx";

// Simple text equation (also available via JSON API above)
new Paragraph({
  children: [
    new MathAPI({
      children: [new MathRun("E = mc²")],
    }),
  ],
});

// Fraction equation (Class API only — complex structures like fractions are not available in JSON)
new Paragraph({
  children: [
    new MathAPI({
      children: [
        new MathFraction({
          numerator: [new MathRun("a")],
          denominator: [new MathRun("b")],
        }),
      ],
    }),
  ],
});
```

::

### Complex Equation

```ts
import {
  Paragraph, Math: MathAPI, MathFraction,
} from "@office-open/docx";

new Paragraph({
  children: [
    new MathAPI({
      children: [
        // x = (a + b) / c
        new MathRun("x"),
        new MathRun("="),
        new MathFraction({
          numerator: [new MathRun("a + b")],
          denominator: [new MathRun("c")],
        }),
      ],
    }),
  ],
});
```

### Math Components

| Component            | Description                        |
| -------------------- | ---------------------------------- |
| `Math`               | Root equation container            |
| `MathRun`            | Text within an equation            |
| `MathFraction`       | Fraction (numerator/denominator)   |
| `MathSubScript`      | Subscript notation                 |
| `MathSuperScript`    | Superscript notation               |
| `MathSubSuperScript` | Combined sub- and superscript      |
| `MathRadical`        | Square root and nth root           |
| `MathSum`            | Summation (∑)                      |
| `MathIntegral`       | Integral (∫)                       |
| `MathLimitLower`     | Lower limit notation               |
| `MathLimitUpper`     | Upper limit notation               |
| `MathFunction`       | Named function (e.g. sin, cos)     |
| `MathMatrix`         | Matrix (rows and columns)          |
| `MathRoundBrackets`  | Parentheses grouping               |
| `MathCurlyBrackets`  | Curly brace grouping               |
| `createMathAccent`   | Accents (hat, bar, tilde, etc.)    |
| `createMathBar`      | Overline/underline bar             |
| `MathEqArr`          | Equation array (aligned equations) |
| `MathBorderBox`      | Border box around expression       |
| `MathBox`            | Box with operator emulation        |
| `MathGroupChr`       | Group character (brace, bracket)   |
| `MathPhant`          | Phantom (invisible placeholder)    |

### MathRun

`MathRun` accepts a string or an options object:

```ts
new MathRun("x");
new MathRun({ text: "x" });
```

### MathFraction

Numerator and denominator accept `MathComponent[]` arrays:

```ts
new MathFraction({
  numerator: [new MathRun("a")],
  denominator: [new MathRun("b")],
  fractionType: "bar", // "bar" (default) | "skw" | "lin" | "noBar"
});
```

### MathRadical

```ts
new MathRadical({
  children: [new MathRun("x + y")], // Radicand
  degree: [new MathRun("3")], // Optional: nth root degree
});
```

### MathSubScript / MathSuperScript / MathSubSuperScript

```ts
new MathSubScript({
  children: [new MathRun("x")],
  subScript: [new MathRun("i")],
});

new MathSuperScript({
  children: [new MathRun("x")],
  superScript: [new MathRun("2")],
});

new MathSubSuperScript({
  children: [new MathRun("x")],
  subScript: [new MathRun("i")],
  superScript: [new MathRun("2")],
});
```

### MathSum / MathIntegral

```ts
new MathSum({
  children: [new MathRun("x")],
  subScript: [new MathRun("i=0")],
  superScript: [new MathRun("n")],
});

new MathIntegral({
  children: [new MathRun("f(x)")],
  subScript: [new MathRun("a")],
  superScript: [new MathRun("b")],
});
```

### MathLimitLower / MathLimitUpper

```ts
new MathLimitLower({
  children: [new MathRun("lim")],
  limit: [new MathRun("x→0")],
});
```

### MathFunction

```ts
new MathFunction({
  name: [new MathRun("sin")],
  children: [new MathRun("x")],
});
```

### MathMatrix

```ts
new MathMatrix({
  rows: [
    [new MathRun("a"), new MathRun("b")],
    [new MathRun("c"), new MathRun("d")],
  ],
});
```

### createMathAccent

Factory function for accents (hat, tilde, etc.):

```ts
import { createMathAccent, MathRun } from "@office-open/docx";

createMathAccent({
  accentCharacter: "^", // "^" for hat, "~" for tilde, etc.
  children: [new MathRun("x")],
});
```

### createMathBar

Factory function for overline/underline bars:

```ts
import { createMathBar, MathRun } from "@office-open/docx";

createMathBar({
  type: "top", // "top" or "bot"
  children: [new MathRun("x")],
});
```

### MathEqArr

Equation array for aligned equations:

```ts
new MathEqArr({
  rows: [
    [new MathRun("a"), new MathRun("="), new MathRun("b + c")],
    [new MathRun("d"), new MathRun("="), new MathRun("e + f")],
  ],
});
```

### MathBorderBox

Draw a border box around an expression with configurable borders and strikethroughs:

```ts
new MathBorderBox({
  children: [new MathRun("a")],
});

// Hide top and bottom borders
new MathBorderBox({
  children: [new MathRun("b")],
  properties: {
    hideTop: true,
    hideBottom: true,
  },
});
```

#### BorderBox Properties

| Property             | Type      | Description                       |
| -------------------- | --------- | --------------------------------- |
| `hideTop`            | `boolean` | Hide top border                   |
| `hideBottom`         | `boolean` | Hide bottom border                |
| `hideLeft`           | `boolean` | Hide left border                  |
| `hideRight`          | `boolean` | Hide right border                 |
| `strikeHorizontal`   | `boolean` | Horizontal strikethrough          |
| `strikeVertical`     | `boolean` | Vertical strikethrough            |
| `strikeDiagonalUp`   | `boolean` | Bottom-left to top-right diagonal |
| `strikeDiagonalDown` | `boolean` | Top-left to bottom-right diagonal |

### MathBox

Wrap content in a box, optionally emulating operator behavior:

```ts
new MathBox({
  children: [new MathRun("x + y")],
  properties: { opEmu: true },
});
```

### MathGroupChr

Add a grouping character (brace, bracket, etc.) above or below content:

```ts
new MathGroupChr({
  children: [
    new MathEqArr({
      rows: [[new MathRun("a")], [new MathRun("b")]],
    }),
  ],
  properties: {
    chr: "{", // Grouping character
    pos: "bot", // Position: "top" or "bot"
    vertJc: "top", // Vertical justification
  },
});
```

### MathPhantom

Create invisible placeholders for spacing control:

```ts
new MathPhant({
    children: [new MathRun("dy")],
    properties: {
        zeroAsc: true, // Zero ascent
        zeroDesc: true, Zero descent
    },
});
```

## Symbol Runs

Insert Wingdings or other symbol font characters inline:

::api-example{type="docx"}

```json [JSON]
{
  "sections": [
    {
      "children": [
        {
          "paragraph": {
            "children": [
              "Arrow: ",
              { "symbolRun": { "char": "F021", "symbolfont": "Wingdings" } },
              " check: ",
              { "symbolRun": { "char": "F052", "symbolfont": "Wingdings" } }
            ]
          }
        }
      ]
    }
  ]
}
```

```ts [Class]
import { Paragraph, TextRun, SymbolRun } from "@office-open/docx";

new Paragraph({
  children: [
    new TextRun("Arrow: "),
    new SymbolRun({ char: "F021", symbolfont: "Wingdings" }),
    new TextRun(" check: "),
    new SymbolRun({ char: "F052", symbolfont: "Wingdings" }),
  ],
});
```

::

### SymbolRun Options

| Option       | Type      | Description                        |
| ------------ | --------- | ---------------------------------- |
| `char`       | `string`  | Hex character code (e.g. `"F021"`) |
| `symbolfont` | `string`  | Font name (default: `"Wingdings"`) |
| `bold`       | `boolean` | Bold                               |
| `italics`    | `boolean` | Italic                             |
| `color`      | `string`  | Hex color code                     |
| `size`       | `number`  | Font size in half-points           |

## Ruby Annotations

Ruby annotations provide pronunciation guides for East Asian text (furigana, pinyin, etc.):

```ts
import { Paragraph, createRuby } from "@office-open/docx";

new Paragraph({
  children: [
    createRuby({
      base: "漢字", // Base text
      text: "かんじ", // Ruby (phonetic) text
    }),
    createRuby({
      base: "汉字",
      text: "hànzì",
      language: "zh-CN", // Language identifier
    }),
  ],
});
```

### Ruby Options

| Property       | Type                            | Description                           |
| -------------- | ------------------------------- | ------------------------------------- |
| `base`         | `string`                        | Base text to annotate                 |
| `text`         | `string`                        | Ruby annotation text                  |
| `alignment`    | `"left" \| "center" \| "right"` | Alignment (default: `"center"`)       |
| `rubyFontSize` | `number`                        | Font size in half-points              |
| `offset`       | `number`                        | Vertical offset in half-points        |
| `language`     | `string`                        | Language identifier (e.g., `"ja-JP"`) |
