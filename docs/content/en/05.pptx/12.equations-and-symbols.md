You can include mathematical equations and special symbols in your presentations.

## Equations in Text Runs

Use Unicode characters and `baseline` for superscript/subscript:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 800,
            "height": 300,
            "textBody": {
              "children": [
                {
                  "children": [
                    { "text": "E = mc", "fontSize": 28 },
                    { "text": "2", "fontSize": 20, "baseline": 30000 }
                  ]
                },
                {
                  "children": [
                    { "text": "H", "fontSize": 28 },
                    { "text": "2", "fontSize": 20, "baseline": -25000 },
                    { "text": "O", "fontSize": 28 }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generatePresentation } from "@office-open/pptx";

generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 800,
            height: 300,
            textBody: {
              children: [
                {
                  children: [
                    { text: "E = mc", fontSize: 28 },
                    { text: "2", fontSize: 20, baseline: 30000 },
                  ],
                },
                {
                  children: [
                    { text: "H", fontSize: 28 },
                    { text: "2", fontSize: 20, baseline: -25000 },
                    { text: "O", fontSize: 28 },
                  ],
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```

## Superscript and Subscript

Use the `baseline` property (in 1000ths of a percent):

```ts
{ text: "x", fontSize: 24 }
{ text: "2", fontSize: 16, baseline: 30000 } // x² (superscript)
{ text: "n", fontSize: 16, baseline: -25000 } // Subscript
```

## Common Math Symbols (Unicode)

```ts
{ text: "π", fontSize: 24 } // pi
{ text: "Σ", fontSize: 24 } // Sigma (sum)
{ text: "∫", fontSize: 24 } // Integral
{ text: "√", fontSize: 24 } // Square root
{ text: "±", fontSize: 24 } // Plus-minus
{ text: "≠", fontSize: 24 } // Not equal
{ text: "≤", fontSize: 24 } // Less than or equal
{ text: "≥", fontSize: 24 } // Greater than or equal
{ text: "→", fontSize: 24 } // Right arrow
{ text: "×", fontSize: 24 } // Multiplication
{ text: "÷", fontSize: 24 } // Division
{ text: "∞", fontSize: 24 } // Infinity
```

## Complex Equation Layout

Build multi-line equations using multiple paragraphs:

::api-example{type="pptx"}

```json [JSON]
{
  "slides": [
    {
      "children": [
        {
          "shape": {
            "x": 50,
            "y": 50,
            "width": 800,
            "height": 350,
            "textBody": {
              "children": [
                {
                  "properties": { "alignment": "CENTER" },
                  "children": [{ "text": "Quadratic Formula", "fontSize": 18, "bold": true }]
                },
                {
                  "properties": { "alignment": "CENTER" },
                  "children": [
                    { "text": "x = (-b ± √(b", "fontSize": 24 },
                    { "text": "2", "fontSize": 16, "baseline": 30000 },
                    { "text": " - 4ac)) / 2a", "fontSize": 24 }
                  ]
                }
              ]
            }
          }
        }
      ]
    }
  ]
}
```

```ts [TypeScript]
import { generatePresentation } from "@office-open/pptx";

generatePresentation({
  slides: [
    {
      children: [
        {
          shape: {
            x: 50,
            y: 50,
            width: 800,
            height: 350,
            textBody: {
              children: [
                {
                  properties: { alignment: "CENTER" },
                  children: [{ text: "Quadratic Formula", fontSize: 18, bold: true }],
                },
                {
                  properties: { alignment: "CENTER" },
                  children: [
                    { text: "x = (-b ± √(b", fontSize: 24 },
                    { text: "2", fontSize: 16, baseline: 30000 },
                    { text: " - 4ac)) / 2a", fontSize: 24 },
                  ],
                },
              ],
            },
          },
        },
      ],
    },
  ],
});
```

::

## Tips

- Use `baseline: 30000` for superscript and `baseline: -25000` for subscript.
- Unicode math symbols render correctly in most presentation viewers.
- For complex mathematical expressions, consider using an image of the equation rendered from LaTeX.
