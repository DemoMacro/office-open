---
title: "Export"
description: "Export documents to buffer, blob, base64, string, or stream"
---

The `Packer` class converts a `Document` into various output formats.

## toBuffer

Returns a `Buffer`. Best for Node.js file I/O.

```ts
import { Packer } from "@office-open/docx";
import { writeFileSync } from "node:fs";

const buffer = await Packer.toBuffer(doc);
writeFileSync("output.docx", buffer);
```

## toBlob

Returns a `Blob`. Best for browser environments.

```ts
const blob = await Packer.toBlob(doc);
const url = URL.createObjectURL(blob);

// Trigger download
const a = document.createElement("a");
a.href = url;
a.download = "output.docx";
a.click();
URL.revokeObjectURL(url);
```

## toBase64String

Returns a Base64-encoded string. Useful for data URLs or API payloads.

```ts
const base64 = await Packer.toBase64String(doc);
const dataUrl = `data:application/vnd.openxmlformats-officedocument.wordprocessingml.document;base64,${base64}`;
```

## toString

Returns a string representation. Useful for debugging.

```ts
const str = await Packer.toString(doc);
console.log(str);
```

## toStream

Returns a Node.js `Readable` stream. Useful for streaming large documents.

```ts
import { createWriteStream } from "node:fs";

const stream = Packer.toStream(doc);
stream.pipe(createWriteStream("output.docx"));
```

## Summary

| Method            | Return Type      | Environment | Use Case        |
| ----------------- | ---------------- | ----------- | --------------- |
| `toBuffer`        | `Buffer`         | Node.js     | File I/O        |
| `toBlob`          | `Blob`           | Browser     | Downloads       |
| `toBase64String`  | `string`         | Any         | Data URLs, APIs |
| `toString`        | `string`         | Any         | Debugging       |
| `toStream`        | `Readable`       | Node.js     | Large files     |
| `toArrayBuffer`   | `ArrayBuffer`    | Any         | Memory handling |
